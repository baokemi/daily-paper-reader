## 摘要
利用大型语言模型（LLMs）作为基于联邦学习（FL）的时间序列基础模型，为在保护私有数据访问的同时，将LLMs的泛化能力迁移到时间序列数据中提供了一种有前景的方法。然而，时间序列数据与现有LLMs以文本为中心的潜在空间之间的语义不匹配常常导致性能下降。同时，现有的FL方法中的参数共享机制，将异构的跨领域时间序列数据建模为统一的连续潜在空间，这与时间序列语义经常表现为离散且重复模式的事实相矛盾。为了解决这些限制，我们提出了\textsc{FeDPM}，一种基于离散原型记忆的时间序列基础模型联邦框架。具体而言，我们为域内时间序列数据学习局部原型记忆先验，然后对跨域记忆进行对齐以促进统一的离散潜在空间，并引入一种特定领域的记忆更新机制，以平衡共享与个性化的原型知识。大量实验表明，\textsc{FeDPM}具有高效性和有效性。代码已在 https://anonymous.4open.science/r/FedUnit-64D1 公开。

## Abstract
Leveraging Large Language Models (LLMs) as federated learning (FL)-based time series foundation models offers a promising way to transfer the generalization capabilities of LLMs to time series data while preserving access to private data. However, the semantic misalignment between time-series data and the text-centric latent space of existing LLMs often leads to degraded performance. Meanwhile, the parameter-sharing mechanism in existing FL methods model heterogeneous cross-domain time-series data into a unified continuous latent space, which contradicts the fact that time-series semantics frequently manifest as discrete and recurring regimes. To address these limitations, we propose \textsc{FeDPM}, a federated framework for time-series foundation models based on discrete prototypical memories. Specifically, we learn local prototypical memory priors for intra-domain time-series data. We then align cross-domain memories to promote a unified discrete latent space and introduce a domain-specific memory update mechanism to balance shared and personalized prototypical knowledge. Extensive experiments demonstrate the efficiency and effectiveness of \textsc{FeDPM}. The code is publicly available at https://anonymous.4open.science/r/FedUnit-64D1.

## 一、研究问题与背景

- **核心问题**：现有将大语言模型（LLMs）应用于联邦学习（Federated Learning, FL）框架下的时间序列基础模型（Time Series Foundation Models, TSFMs）时，存在两个主要挑战：
  1. **语义错配问题**：LLMs 的潜在表示空间以文本为中心，与时间序列的动态模式存在语义不一致，导致建模性能下降。
  2. **连续表示假设失效**：传统 FL 参数共享机制假定所有域的时间序列能嵌入到统一的连续空间，但实际时间序列语义常呈现离散、周期性或相变行为（如交通流状态转变）。

- **研究动机**：希望在保护数据隐私的前提下，跨域共享时间序列的共性知识，通过构造**离散原型表示（Discrete Prototypical Representations）**改善时序语义表达与跨域泛化能力。

- **总体目标**：建立一种兼顾隐私与效率的联邦时间序列基础模型框架，提升不同域间的知识迁移与语义一致性。

## 二、方法论：FeDPM 框架

### 核心思想
- 通过引入**离散原型记忆（Discrete Prototypical Memories）**替代传统连续参数共享，实现跨域时间序列的**语义量化与聚合**。
- 每个客户端（域）学习本地的原型记忆先验，用于表示该域典型的时序模式；服务器执行原型对齐与更新，以形成共享与个性化的记忆结构。

### 模型结构与模块
1. **本地记忆学习（Local Prototypical Memory Priors）**
   - 每个客户端将时间序列切分为若干非重叠的帧（patch）。
   - 经局部编码器（Encoder）得到 patch 表征向量。
   - 通过**原型检索（Prototypical Memory Retrieval, PMR）**机制，将连续表征映射为最近的离散原型，从而实现量化。
   - 原型集合记为 \( P_n = \{ e_{n,1},...,e_{n,M} \} \)，形成本地记忆库。

2. **跨域记忆对齐与更新（Cross-Domain Memory Update）**
   - 服务器聚合客户端上传的原型集合，计算跨域原型的余弦相似矩阵：
     \[
     s_{i,j}^{m,n} = \frac{e_{m,i}^\top e_{n,j}}{\|e_{m,i}\|\|e_{n,j}\|}
     \]
   - 通过阈值 \( \delta \) 筛选相似原型，构建原型图，用 BFS 查找语义聚类。
   - 对每个聚类求均值得到共享原型中心：
     \[
     e_s = \frac{1}{|I_s|}\sum_{i \in I_s} e_i
     \]
   - 保留部分共享原型（比例 \( \gamma \)），其余由**效用–多样性评分**挑选出有代表性的个性化原型。

3. **优化目标**
   - 综合预测误差与原型量化约束：
     \[
     \mathcal{L} = \mathcal{L}_{Pred} + \beta \mathcal{L}_{ME} + \mathcal{L}_{MC}
     \]
     其中使用 Smooth L1 损失与 VQ-VAE 风格的停梯度项。

4. **通信机制**
   - 客户端与服务器仅交换原型记忆，而非完整模型权重，大幅减少通信量。

5. **伪算法流程**
   - 每轮：客户端局部训练 → 上传记忆与使用频率 → 服务器执行原型对齐 + 共享/个性化记忆更新 → 下发更新记忆。

## 三、实验设计

### 数据集与场景
- 共使用 **7 个真实世界时间序列数据集**：
  - ETTh1, ETTh2, ETTm1, ETTm2（电力设备监控）
  - Electricity（电力消耗）
  - Weather（气象预测）
  - Exchange（汇率变动）
- 每个数据集对应一个 FL 客户端，形成跨域分布式学习。

### 对比基线
分三个类别：
1. **联邦基础模型（FL-FM）**：Time-FFM、FFTS、FL-iTransformer、FL-PatchTST。
2. **集中式基础模型（Cen-FM）**：TOTEM、UniTime、Cen-PatchTST。
3. **集中式专家模型（Expert）**：TimesNet、DLinear、FEDformer、iTransformer、PatchTST。

### 评价指标
- 使用 MSE 和 MAE 度量预测精度；
- 对比通信参数量与可训练参数数量；
- 额外测试**few-shot 设置（5%、10%样本）**下的模型泛化性能。

## 六、主要结论与发现

- **性能提升**：FeDPM 在全部基线中获得最多的首位性能排名，平均 MAE 优于强基线 FFTS 约 4.9%。
- **通信效率**：仅上传记忆单元，通信参数量减少 97.03%，训练参数量减少 20.37%。
- **泛化能力**：在少样本实验中仍维持优异表现，MAE 分别降低 7.29% 和 6.42%。
- **隐私鲁棒性**：在注入三种随机噪声机制后仍保持稳定性能，验证其隐私可适应性。
- **语义表示质量**：原型可区分不同时间序列结构，t-SNE 可视化显示良好分离度与可解释性。

## 七、优点与创新点

- **表示创新**：首次引入离散记忆原型在联邦时间序列模型中进行跨域语义聚合。
- **通信优化**：通过“记忆交换”替代“参数同步”，极大降低通信成本。
- **跨域可解释性**：原型聚类使得不同域间共享模式更直观、便于解释。
- **普适性强**：模型架构可兼容多种编码器（Transformer、CNN、RNN等）。
- **隐私与性能平衡**：实验验证加入噪声后性能几乎不下降，具备实用潜力。
