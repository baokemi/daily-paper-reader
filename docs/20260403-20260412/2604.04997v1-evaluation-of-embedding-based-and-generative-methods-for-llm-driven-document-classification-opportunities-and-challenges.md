## 摘要
本研究对基于嵌入的方法和生成式模型在地球科学技术文档分类中的性能进行了对比分析。通过使用一个多学科基准数据集，我们评估了模型在准确性、稳定性和计算成本之间的权衡。研究发现，结合了Chain-of-Thought（CoT）提示的生成式视觉-语言模型（VLM），如Qwen2.5-VL，在零样本分类中取得了优于当前先进多模态嵌入模型（如QQMM）的准确率（82% 对比 63%）。此外，我们还表明，尽管监督微调（SFT）可以提升VLM的性能，但其对训练数据的不平衡较为敏感。

## Abstract
This work presents a comparative analysis of embedding-based and generative models for classifying geoscience technical documents. Using a multi-disciplinary benchmark dataset, we evaluated the trade-offs between model accuracy, stability, and computational cost. We find that generative Vision-Language Models (VLMs) like Qwen2.5-VL, enhanced with Chain-of-Thought (CoT) prompting, achieve superior zero-shot accuracy (82%) compared to state-of-the-art multimodal embedding models like QQMM (63%). We also demonstrate that while supervised fine-tuning (SFT) can improve VLM performance, it is sensitive to training data imbalance.

## 1. 研究核心问题与整体含义

- **研究动机**：  
  地球科学（尤其是油气行业）积累了大量非结构化技术文档，如地质报告、测井曲线、地震剖面等。手动对这些文档分类是一项耗时且主观的工作，自动化分类可以极大提高信息组织与检索效率。  
- **核心问题**：  
  传统文本模型无法处理多模态信息（图像、布局、文本混合），同时 OCR 的质量不稳定，导致纯文本模型性能受限。因此亟需能处理图文混合内容的大模型框架。  
- **研究目标**：  
  系统比较两类主流方法：
  1. **嵌入式方法（embedding-based）**：将文档转化为向量，以相似度投票方式进行分类。  
  2. **生成式方法（generative, Vision-Language Models, VLMs）**：直接使用多模态大模型生成类别标签。  
  论文旨在量化两者在**准确率、稳定性、计算成本**上的权衡。

## 2. 方法论：整体思路与关键技术

### 2.1 嵌入式分类（Embedding-based Voting）

- **思路**：将文档图像和类别定义分别编码为高维向量，通过余弦相似度投票判定类别。  
- **技术流程**：
  1. 将文档第一页转化为图像（按最大边长 8192 像素缩放）。  
  2. 使用多模态嵌入模型生成文档向量；类标签也通过“带领域定义”的提示生成类向量。  
  3. 计算文档与每个类向量的余弦相似度，取最大者为预测结果。  
- **指标设计**：
  - 使用宏平均 F1 和整体准确率。  
  - 采用聚类度量（如内部紧度 \(L_{intra}\)、类间分离度 \(L_{inter}\)、Silhouette、DB Index、CH Index 等）评价嵌入空间质量。  

### 2.2 生成式视觉语言模型（Generative VLM）

- **思路**：VLM 直接通过生成推理输出类别标签，无需向量匹配。
- **技术设定**：
  - 采用两种提示（prompt）策略：  
    - **Base**：带人格设定的基本提示。  
    - **Plus**：结合 Chain-of-Thought（CoT）逐步推理与领域知识指导。  
  - 输入为文档图像与提示语，模型输出类标签并进行解析。

### 2.3 监督微调（Supervised Fine-tuning, SFT）

- **实施**：  
  使用 Qwen2.5-VL-7B 模型，在约 7000 条数据上进行领域微调。  
  - 数据分布极不均衡，其中部分类别仅有几十样本。  
  - 为防止提示过拟合，构造了多种模板组合用于训练样本生成。  
- **目标**：评估领域适配对分类能力的提升及对数据均衡性的敏感性。
