## 摘要
测试时扩展（Test-time scaling）研究探讨固定的扩散语言模型（Diffusion Language Model, DLM）在推理时给予更多计算量而无需额外训练，是否能够生成更优输出。然而，朴素的 best-of-K 采样方法存在根本局限，因为它反复从相同的基础扩散分布中抽样，而该分布的高概率区域往往与高质量输出不匹配。我们提出了 $S^3$（Stratified Scaling Search，分层扩展搜索），一种由验证器引导的经典搜索方法，它通过在去噪过程中重新分配计算资源而非仅在最终输出阶段进行，从而改进生成质量。在每一步去噪中，$S^3$ 扩展多个候选轨迹，利用轻量且无需参考的验证器对其进行评估，选择性地重采样具有潜力的候选，同时保持搜索前沿的多样性。该过程有效近似了一种奖励倾斜的采样分布，使其偏向高质量输出，同时仍锚定于模型先验。我们在 LLaDA-8B-Instruct 模型上对 MATH-500、GSM8K、ARC-Challenge 和 TruthfulQA 数据集进行了实验，结果表明，$S^3$ 在各项基准中均稳步提升性能，尤其在数学推理任务上取得最大增益，同时无需更改底层模型或解码调度。这些结果表明，对去噪轨迹进行经典搜索为 DLM 的测试时扩展提供了一种切实可行的机制。

## Abstract
Test-time scaling investigates whether a fixed diffusion language model (DLM) can generate better outputs when given more inference compute, without additional training. However, naive best-of-$K$ sampling is fundamentally limited because it repeatedly draws from the same base diffusion distribution, whose high-probability regions are often misaligned with high-quality outputs. We propose $S^3$ (Stratified Scaling Search), a classical verifier-guided search method that improves generation by reallocating compute during the denoising process rather than only at the final output stage. At each denoising step, $S^3$ expands multiple candidate trajectories, evaluates them with a lightweight reference-free verifier, and selectively resamples promising candidates while preserving diversity within the search frontier. This procedure effectively approximates a reward-tilted sampling distribution that favors higher-quality outputs while remaining anchored to the model prior. Experiments with LLaDA-8B-Instruct on MATH-500, GSM8K, ARC-Challenge, and TruthfulQA demonstrate that $S^3$ consistently improves performance across benchmarks, achieving the largest gains on mathematical reasoning tasks while leaving the underlying model and decoding schedule unchanged. These results show that classical search over denoising trajectories provides a practical mechanism for test-time scaling in DLMs.

## 一、研究动机与问题背景

- **背景**：  
  扩散语言模型（Diffusion Language Models, DLMs）是一类通过逐步去噪生成文本的生成式模型，已在性能上接近甚至超过同规模自回归语言模型。  
  随着模型规模增长受限，学界开始探索“测试时扩展”（test-time scaling）——即在不重新训练的前提下，通过增加推理时计算量提升性能。

- **问题**：  
  现有的推理时扩展方法（如 *best-of-K sampling*）会多次从相同的基础分布 \(p_0(x)\) 采样，这种分布的高概率区域往往与“高质量输出”不对齐。  
  这一“密度–质量失配”（density–quality mismatch）导致简单增加样本数量只能带来 **对数级的效益增长**，无法有效利用计算资源。

- **研究目的**：  
  作者希望通过重新分配推理计算资源，使扩散解码过程中的每一步都能利用轻量验证器指导搜索，从而近似一种“奖励倾斜（reward-tilted）”的理想采样分布，以更高效地探索高质量输出区域。

## 二、方法论：$S^3$（Stratified Scaling Search）

### 1. 核心思想
- 不仅在最终输出阶段进行选择，而是在**每个去噪步骤**上：
  - 扩展多个候选轨迹；
  - 用轻量验证器（verifier）估计未来潜在质量；
  - 利用采样与重采样机制集中计算资源在前景较好的轨迹上；
  - 同时保持一定的样本多样性，避免模式坍塌。

- 理论基础：  
  在 KL 约束下的最优推理目标是奖励倾斜分布  
  \[
  \tilde{p}_0(x) \propto p_0(x) e^{\tau f(x)}
  \]
  其中 \(f(x)\) 是验证器打分函数，\(\tau\) 为温度系数。  
  但直接采样难以实现，作者提出在扩散轨迹空间上近似这一目标。

### 2. 方法流程（文字化算法）
- **设定**：输入扩散模型 \(p_\theta\)，验证器 \(f\)，固定解码调度 \(e=(K, \pi, T)\)，粒子数 \(N\)，分支系数 \(b\)，权重温度 \(\lambda\)。
- **流程**：
  1. **初始化**：在完全mask的起点 \(x_T\) 上建立 \(N\) 个粒子。  
  2. **逐步去噪（每个时间步 t）**：
     - 每个粒子扩展 \(b\) 个候选 \(x_{t-1}^{(i,j)}\)，由模型条件分布采样；
     - 对每个候选，计算其“一步清理预测”（one-step clean prediction）\(\hat{x}_0^{(i,j,t)}\)，并用验证器得分 \(s_{i,j,t}=f(\hat{x}_0^{(i,j,t)})\)；
     - 依据权重 \(w_{i,j,t}=\exp(\lambda s_{i,j,t})\) 进行重采样，使用 **Srinivasan Sampling Process (SSP)** 保持多样性；
  3. **输出选择**：对最终 \(N\) 个解码结果采用**多数投票**，若有平局则以负对数似然（NLL）最低者为准。

### 3. 技术要点
- **SSP重采样**：低方差、依赖性取整的粒子采样方法，可防止搜索早期坍塌。
- **轻量验证器**：不依赖标注数据或LLM裁判，基于生成文本的自检指标：
  - 结构完整性、算术一致性、答案可追溯性、模型置信度、非退化性；
  - 对于特定任务（数学、常识问答、多选、Sudoku等）会有任务约束项 \(s_{\text{constraint}}\)；
  - 整体验证器输出为线性加权综合分数 \(f(x) = \sum_k \alpha_k s_k(x) + \alpha_c s_{\text{constraint}}(x)\)。
