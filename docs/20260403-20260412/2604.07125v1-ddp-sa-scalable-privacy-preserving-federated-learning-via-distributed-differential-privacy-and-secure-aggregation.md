## 摘要
本文提出了 DDP-SA，这是一种可扩展的隐私保护联邦学习框架，结合了客户端的本地差分隐私（Local Differential Privacy, LDP）与全阈值加法秘密共享（Additive Secret Sharing, ASS）以实现安全聚合。不同于仅依赖差分隐私或安全多方计算（Secure Multi-Party Computation, MPC）的现有方法，DDP-SA 融合了这两种技术，在保持计算可行性的同时提供更强的端到端隐私保障。该框架引入了双阶段保护机制：客户端首先通过校准的拉普拉斯噪声扰动其本地梯度，然后将扰动后的梯度分解为加法秘密份额，分发到多个中间服务器。此设计确保：(i) 任何单一被攻陷的服务器或通信信道都无法泄露关于单个客户端更新的信息；(ii) 参数服务器仅能重建聚合后的噪声梯度，而永远无法获得任何客户端的具体贡献。大量实验结果表明，DDP-SA 相较于单纯的 LDP 能显著提升模型精度，同时提供比仅用 MPC 更强的隐私保护。所提出的框架在参与者数量方面呈线性扩展性，并以可控的计算与通信开销，为联邦学习应用提供了一种实用的隐私保护解决方案。

## Abstract
This article presents DDP-SA, a scalable privacy-preserving federated learning framework that jointly leverages client-side local differential privacy (LDP) and full-threshold additive secret sharing (ASS) for secure aggregation. Unlike existing methods that rely solely on differential privacy or on secure multi-party computation (MPC), DDP-SA integrates both techniques to deliver stronger end-to-end privacy guarantees while remaining computationally practical. The framework introduces a two-stage protection mechanism: clients first perturb their local gradients with calibrated Laplace noise, then decompose the noisy gradients into additive secret shares that are distributed across multiple intermediate servers. This design ensures that (i) no single compromised server or communication channel can reveal any information about individual client updates, and (ii) the parameter server reconstructs only the aggregated noisy gradient, never any client-specific contribution. Extensive experiments show that DDP-SA achieves substantially higher model accuracy than standalone LDP while providing stronger privacy protection than MPC-only approaches. The proposed framework scales linearly with the number of participants and offers a practical, privacy-preserving solution for federated learning applications with controllable computational and communication overhead.

## 一、核心问题与研究动机

- **研究背景**  
  联邦学习（Federated Learning, FL）允许多个客户端在保留本地数据的情况下，共同训练一个全局模型，从而避免数据集中化的隐私风险。然而，已有研究表明，即使只共享模型梯度或参数，仍然可能通过推理攻击泄露用户的敏感信息（如属性、样本重建等）。

  2. **安全多方计算（MPC）**：可保证服务端仅获得聚合结果，但计算与通信开销巨大，难以扩展到大规模场景。

- **研究目标**  
  论文的核心目标是：  
  > 设计一种兼具强隐私保护与高可扩展性的联邦学习框架，同时保持可接受的准确率与计算开销。

## 二、方法论：DDP-SA 框架设计

### 1. 核心思想  
DDP-SA（Distributed Differential Privacy via Secure Aggregation）融合了：
- **本地差分隐私 (LDP)**：客户端在上传梯度前添加噪声；
- **加性秘密共享 (Additive Secret Sharing, ASS)**：通过将扰动后的梯度分成多个份额并分发到多个中间服务器，实现安全聚合。

> 这种“双层保护机制”在数学上保持原有的（ϵ, δ）差分隐私保证，同时通过密码学手段增强通信和聚合过程中的隐私安全。

### 2. 架构组成
系统共分三类节点：
- **客户端**：计算本地梯度 → 添加拉普拉斯噪声 → 生成秘密份额并分发。
- **中间服务器**：聚合来自所有客户端的分享份额 → 汇总结果发送至参数服务器。
- **参数服务器**：重建全局聚合梯度 → 更新并广播模型参数。

> 与传统双层架构相比，DDP-SA增加了中间服务器层，实现线性扩展。

### 3. 主要算法流程（简述）
**算法 2 — DDP-SA**
1. 参数服务器广播当前模型参数。  
2. 每个客户端执行：
   - 计算局部梯度并进行 **ℓ1 范数剪裁**；
   - 添加拉普拉斯噪声（噪声尺度 ∼ Δ/ϵ）；
   - 固定精度编码并生成 **m 份加性秘密共享**；
   - 将每份份额发送至不同的中间服务器。  
3. 中间服务器对收到的份额求和后发送至参数服务器。  
4. 参数服务器重建总梯度，进行模型更新。  
5. 重复以上过程至收敛。

### 4. 理论分析
- 单轮保证：满足（ϵ, δ）-DP，无额外隐私损失。  
- 多轮训练：通过基本与高级组合定理分析隐私预算的累积，支持长程训练隐私管理。  
- 安全模型：假设为半诚实参与者；若攻击者控制少于全部中间服务器，则不能恢复任何单一客户端的梯度。
