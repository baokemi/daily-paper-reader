## 摘要
离散扩散语言模型已成为自回归语言模型的有力替代方案，但在参数与显存预算受限的条件下高效训练仍然具有挑战性。现代架构主要基于全词表的token预测层，这部分占据了相当比例的模型参数（例如，在小规模的DiT式设计中超过20%），并且通常主导着GPU显存的峰值使用。这导致在有限训练资源下参数与显存利用效率低下。为了解决这一问题，我们重新审视了显式全词表预测的必要性，转而利用token之间的固有结构，构建树状结构的扩散语言模型。具体而言，我们将扩散过程建模为对应于预构建词汇树中一个token的祖先节点的中间潜在状态。该树状结构的分解使分类维度呈指数级下降，使预测头大小可忽略不计，并使参数能够重新分配至加深Attention模块。实验结果表明，在相同的参数预算下，我们的方法将GPU显存峰值使用减半，同时在困惑度(perplexity)性能上与目前最先进的离散扩散语言模型相当。

## Abstract
Discrete diffusion language models have emerged as a competitive alternative to auto-regressive language models, but training them efficiently under limited parameter and memory budgets remains challenging. Modern architectures are predominantly based on a full-vocabulary token prediction layer, which accounts for a substantial fraction of model parameters (e.g., more than 20% in small scale DiT-style designs) and often dominates peak GPU memory usage. This leads to inefficient use of both parameters and memory under constrained training resources. To address this issue, we revisit the necessity of explicit full-vocabulary prediction, and instead exploit the inherent structure among tokens to build a tree-structured diffusion language model. Specifically, we model the diffusion process with intermediate latent states corresponding to a token's ancestor nodes in a pre-constructed vocabulary tree. This tree-structured factorization exponentially reduces the classification dimensionality, makes the prediction head negligible in size, and enables reallocation of parameters to deepen the attention blocks. Empirically, under the same parameter budget, our method reduces peak GPU memory usage by half while matching the perplexity performance of state-of-the-art discrete diffusion language models.

## 1. 研究背景与核心问题

- **研究动机**：  
  当前的离散扩散语言模型（DLM, Discrete Diffusion Language Model）已成为自回归语言模型的有力替代，但仍面临**训练效率低、显存占用高**的问题。特别是在较小规模模型下，全词表预测层占据了 **12%–20% 以上模型参数**，同时主导了训练过程中的峰值 GPU 显存使用。
- **核心问题**：  
  现有 DLM 采用“直接预测整个词表”的方式，忽略了词汇间固有的层次结构与语义聚类特性，导致内存浪费与参数重复。
- **研究目标**：  
  作者希望通过构建一种能利用词汇层次结构的模型，使得在**相同参数预算下提高训练效率、降低显存开销**，同时不牺牲语言建模性能。

## 2. 方法论

### 2.1 核心思想

- 提出 **Tree-Structured Diffusion Language Model (TDLM)**，将传统 DLM 的扁平化 token 预测改为基于词汇树的**层级化子节点预测**。
- 模型不再直接预测完整词表，而是在一个预构建的词汇树上进行**逐层“child prediction”**：
  - 通过词汇树层次，将 token 预测分解为从上到下的系列决策；
  - 每一步仅在少量候选子节点中分类，从而**指数级减少输出维度**。

### 2.2 技术细节

- **树结构构造**：
  - 使用递归 K-means 对预训练 token 嵌入进行聚类；
  - 每个节点分为 K 个子节点直至叶节点，每个叶节点对应一个真实 token；
  - 保持所有叶节点深度一致。
  
- **扩散建模框架**：
  - 将传统离散扩散过程分解为若干“层内过程”（in-level process），每一层对应词汇树的一个高度；
  - 在连续时间马尔可夫链（CTMC）框架下建模，每层都有独立的前向与反向过程；
  - **前向扩散**：在同层节点之间逐步从子节点混合到父节点；
  - **反向生成**：由当前节点预测其子节点分布，而非直接预测最终 token；
  - 定义新的 **ELBO（证据下界）公式** 来适配这一分层 CTMC 模型。

- **参数与计算效率**：
  - 传统输出层规模为 `d × V`（例如 768×50k=38.4M 参数）；
  - TDLM 仅需 `d × K`，若 `K=512`，仅为 0.4M 参数；
  - 参数节省可用于增加注意力层数（模型更深）。

### 2.3 算法与流程概述

1. 初始化词汇树；
2. 对每个 token，根据扩散阶段确定其所在层；
3. 在该层预测其真实子节点；
4. 基于推导的层内 ELBO 优化目标训练模型；
5. 跨层的总目标为所有层 ELBO 之和。

## 6. 主要结论与发现

- **性能表现**：
  - 在相同训练预算下，TDLM 的验证困惑度与 HDLM 接近（甚至在 base 规模略优）；
  - 相对 MDLM 和 GIDD 均显著提升（≈1.5–2 PPL）。
- **效率表现**：
  - 峰值 GPU 显存占用减少 **约 50%**；
  - 训练吞吐量提升 **约 25%–30%**；
  - 使得小规模模型在有限资源（4×24GB GPU）上亦能有效训练。
- **结构规律发现**：
  - 浅树（较大分支因子）能获得更低负 ELBO；
  - 层级越高（靠近根），学习难度越大；
  - 为层级加权训练并无显著提升，表明默认学习目标已较优化。

## 8. 局限与不足

- **性能差距**：  
  小规模模型（TDLM-small）在验证困惑度上略逊于 HDLM，说明浅层树结构在小网络中未充分发挥。
- **树结构依赖性**：  
  模型性能对树的构造（K、聚类质量）较敏感；
  层数过深（如二叉树）时，训练收敛较慢。
- **任务范围单一**：  
  所有实验均基于语言建模任务（OWT），未测试下游任务（如生成质量、few-shot能力）。
- **尚未评估推理延迟与参数共享方案**：  
  尽管训练显存减少，但推理效率和分布式可扩展性未深入分析。
