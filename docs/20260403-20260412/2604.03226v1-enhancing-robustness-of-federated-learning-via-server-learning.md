## Abstract
This paper explores the use of server learning for enhancing the robustness of federated learning against malicious attacks even when clients' training data are not independent and identically distributed. We propose a heuristic algorithm that uses server learning and client update filtering in combination with geometric median aggregation. We demonstrate via experiments that this approach can achieve significant improvement in model accuracy even when the fraction of malicious clients is high, even more than $50\%$ in some cases, and the dataset utilized by the server is small and could be synthetic with its distribution not necessarily close to that of the clients' aggregated data.

## 1. 核心问题与研究背景

- **研究动机**  
  联邦学习（Federated Learning, FL）允许多客户端在保持数据本地的情况下共同训练全局模型，但在实际场景中面临两大挑战：
  1. 客户端数据往往**非独立同分布（Non-IID）**，导致模型全局聚合后表现不稳定；
  2. **拜占庭攻击（Byzantine attacks）**等恶意行为可能破坏模型学习，如客户端上传被污染或伪造的更新。

- **总体目标**  
  提出一种能够在客户端数据异质且恶意客户端比例较高情况下，仍然保持良好训练性能的强鲁棒性联邦学习框架。

- **核心思想**  
  借助**服务器端学习（Server Learning, SL）**提升鲁棒性，将服务器自身的数据（可为小规模、甚至合成数据）用于：
  1. 识别潜在恶意更新（过滤功能）；  
  2. 提供正向学习引导（学习功能）；  
  3. 与几何中值聚合算法结合，提高整体稳健性。

## 2. 方法论：核心思想与技术细节

### 2.1 总体框架
算法命名为 **RoFSL (Robust Federated Learning via Server Learning)**，将服务器学习、过滤机制和鲁棒聚合三部分整合。

### 2.2 数学建模
目标优化问题定义为：
\[
\min_{x \in \mathbb{R}^d} \gamma f_0(x) + \sum_{i \in [N]\setminus B} p_i f_i(x)
\]
其中：
- \( f_0(x) \)：服务器数据损失函数；
- \( f_i(x) \)：客户端损失函数；
- \( B \)：恶意客户端集合；
- \( \gamma \)：控制服务器学习权重的超参数；
- \( p_i = n_i / n \)：数据权重。

服务器数据集可以是小规模、非同分布或合成数据。

### 2.3 算法流程（文字说明）
1. **客户端更新阶段**  
   各客户端执行 K 次局部 SGD 更新后上传模型更新至服务器。  
   恶意客户端可以返回任意错误更新。
2. **服务器端过滤**  
   两种过滤机制：
   - **角度过滤（Angle-based Filter, AF）**：计算客户端更新与服务器梯度的余弦相似度，剔除偏离方向过大的更新；
   - **损失过滤（Loss-based Filter, LF）**：计算近似下降分数 \( sc_\rho(i) = -\langle \Delta x_i, \nabla f_0(x) \rangle - \rho \|\Delta x_i\|^2 \)，筛除分数最低的部分客户端。
3. **鲁棒聚合**  
   使用 **几何中值（Geometric Median）** 聚合过滤后的客户端模型，保证至多半数恶意更新仍可稳健。
4. **服务器学习**  
   服务器在自身数据上继续局部更新完善模型（避免伪梯度放大），权重由 \(\gamma\) 控制。
5. **剪切与限制**  
   使用 **Norm Clipping** 控制聚合模型的更新幅度，防止少数异常值影响全局模型。

## 6. 主要结论与发现

- **核心结论**
  - 服务器学习（SL）配合过滤与几何中值聚合显著提升鲁棒性；
  - 当恶意客户端比例高达 0.6 时（>50%），仍能保持稳定训练与较高准确率；
  - 损失过滤（LF）优于角度过滤（AF），且参数对结果不敏感；
  - 适度提高服务器学习权重（γ ∈ [0.05, 1]）可以抵消数据异质和攻击带来的影响。

- **现象观察**
  - 单独使用过滤或服务器学习均不足以对抗严重攻击；
  - 组合策略可使系统获得“有效诚实多数”效果；
  - 服务器数据分布偏差不显著影响稳定性，说明方法对服务器数据质量要求低。
