## 摘要
跨域推荐（Cross-domain Recommendation, CDR）利用多域之间的相关性来缓解数据稀疏问题。作为该领域的核心任务，跨域推荐主要关注预测那些在源域中有交互但在目标域中缺乏行为记录的用户偏好。现有方法主要依靠重叠用户作为知识迁移的锚点。然而在现实场景中，重叠用户往往稀少，大多数用户仅在单一域中存在交互。对于这些用户，缺乏显式对齐信号使得细粒度偏好迁移本质上变得困难。为了解决这一挑战，本文提出了用于跨域推荐的语言引导条件扩散框架（Language-Guided Conditional Diffusion for CDR, LGCD），该框架将大型语言模型（LLMs）与扩散模型结合用于跨域序列推荐。具体来说，我们利用LLM的推理能力弥合域间差距，通过为单域用户推断潜在的目标偏好并将其映射到真实物品，从而构造伪重叠数据。我们区分真实与伪交互路径，并引入附加的监督约束以减轻伪交互带来的语义噪声。此外，我们设计了一个条件扩散架构，以基于源域模式精确引导目标用户表示的生成。大量实验证明，LGCD在跨域推荐任务中显著优于现有的最先进方法。

## Abstract
Cross-domain Recommendation (CDR) exploits multi-domain correlations to alleviate data sparsity. As a core task within this field, inter-domain recommendation focuses on predicting preferences for users who interact in a source domain but lack behavioral records in a target domain. Existing approaches predominantly rely on overlapping users as anchors for knowledge transfer. In real-world scenarios, overlapping users are often scarce, leaving the vast majority of users with only single-domain interactions. For these users, the absence of explicit alignment signals makes fine-grained preference transfer intrinsically difficult. To address this challenge, this paper proposes Language-Guided Conditional Diffusion for CDR (LGCD), a novel framework that integrates Large Language Models (LLMs) and diffusion models for inter-domain sequential recommendation. Specifically, we leverage LLM reasoning to bridge the domain gap by inferring potential target preferences for single-domain users and mapping them to real items, thereby constructing pseudo-overlapping data. We distinguish between real and pseudo-interaction pathways and introduce additional supervision constraints to mitigate the semantic noise brought by pseudo-interaction. Furthermore, we design a conditional diffusion architecture to precisely guide the generation of target user representations based on source-domain patterns. Extensive experiments demonstrate that LGCD significantly outperforms state-of-the-art methods in inter-domain recommendation tasks.

## 一、研究动机与背景

- **研究问题**：跨域推荐（Cross-Domain Recommendation, CDR）旨在利用多个相关领域的用户行为数据，缓解单一领域中数据稀疏和冷启动问题。  
- **核心挑战**：  
  - 传统跨域推荐依赖“重叠用户”（即同时存在于源域与目标域的用户）作为知识迁移的桥梁。  
  - 实际场景中，**重叠用户极少**，绝大多数人只在单一域有行为记录。  
  - 对于这些“非重叠用户”，缺乏显式对齐信号，无法学习跨域偏好映射，导致冷启动推荐困难。  
- **研究意义**：论文提出的 **LGCD（Language-Guided Conditional Diffusion）** 框架，尝试利用大语言模型（LLM）的语言推理能力与扩散模型（Diffusion Model）的生成建模能力，实现单域用户的跨域偏好迁移，是一次**“语言推理 + 生成建模”融合的跨域推荐探索**。

## 二、方法论与核心思想

### 1. 整体思路

- 将跨域推荐任务建模为**条件生成问题**：在已知源域用户行为的条件下，生成其在目标域的潜在偏好表示。
- 引入 **大语言模型（LLM）** 推理潜在跨域兴趣，生成**伪重叠数据**；
- 利用 **条件扩散模型**，在源域信号引导下生成精确的目标域用户表示；
- 通过**解耦训练与监督机制**，避免由伪数据引入的语义噪声。

### 2. 模型结构

LGCD 框架包含三个主要模块：

#### （1）LLM-Based Pseudo-Interaction Generation（LPG）
- 输入用户在源域的交互序列文本描述；
- 通过 LLM（例如 Baichuan2-7B-Chat）生成目标域中可能感兴趣的项目描述；
- 再通过语义检索（使用 jina-embeddings-v2-base-en 文本嵌入）将生成文本映射至**真实目标域物品**；
- 形成“伪重叠交互序列”，作为目标域偏好的语义线索。

#### （2）Conditional Diffusion Preference Generator（CDPG）
- 在伪重叠序列的基础上，用扩散模型学习源域→目标域的映射；
- 设置双路径训练策略：  
  - 对真实重叠用户路径，使用真实目标域数据直接监督；  
  - 对伪重叠用户路径，引入 **对齐约束（L_align）** 防止噪声扩散；
- 采用 **交叉注意力（Cross-Attention）** 机制在反向扩散过程中逐步注入源域偏好；
- 引入 **Guesser Network** 作为先验预测器，在推理阶段提供目标语义空间的初始向量，减轻纯噪声生成的不稳定性。

#### （3）Mixture-of-Experts（MoE） Fusion Module
- 将源域条件信号与扩散生成的目标域表示通过专家网络融合；
- 利用门控网络自适应分配专家权重，使融合结果兼顾源域可迁移模式和目标域特定特征；
- 输出最终的用户表示，用于目标域项目推荐。

### 3. 优化目标与损失函数
- **扩散重建损失**：\( L_{diff} \)，使去噪过程逼近真实目标特征；
- **猜测网络损失**：\( L_{guess} \)，用于目标域先验学习；
- **对齐约束损失**：\( L_{align} \)，减少伪数据偏移；
- **推荐损失**：\( L_{rec} \)，优化最终推荐精度；
- **总体损失**：  
  - 重叠用户：\( L_{total} = L_{diff} + L_{guess} + L_{rec} \)  
  - 非重叠用户：\( L_{total} = L_{diff} + L_{guess} + L_{align} + L_{rec} \)
