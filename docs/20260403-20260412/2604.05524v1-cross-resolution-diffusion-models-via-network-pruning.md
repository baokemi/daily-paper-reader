## 摘要
扩散模型在图像合成方面展现出令人印象深刻的性能，但许多基于 UNet 的模型通常在固定分辨率下进行训练。当在非训练分辨率下生成图像时，它们的质量往往会下降。我们将这一问题归因于分辨率相关的参数行为——在默认分辨率下表现良好的权重在空间尺度变化时可能变得不利，从而削弱语义对齐并导致 UNet 结构的不稳定。基于这一分析，本文提出了 CR-Diff，这是一种通过剪枝扩散模型部分参数来改进跨分辨率视觉一致性的全新方法。具体而言，CR-Diff 包括两个阶段：首先进行分块剪枝以选择性地去除不利权重；随后执行剪枝输出放大以进一步纯化剪枝后的预测。实证研究表明，CR-Diff 能在多种扩散模型骨架和未见分辨率下提升感知保真度与语义一致性，同时在默认分辨率上保持较高性能。此外，CR-Diff 支持特定提示的细化，能够实现按需的质量增强。

## Abstract
Diffusion models have demonstrated impressive image synthesis performance, yet many UNet-based models are trained at certain fixed resolutions. Their quality tends to degrade when generating images at out-of-training resolutions. We trace this issue to resolution-dependent parameter behaviors, where weights that function well at the default resolution can become adverse when spatial scales shift, weakening semantic alignment and causing structural instability in the UNet architecture. Based on this analysis, this paper introduces CR-Diff, a novel method that improves the cross-resolution visual consistency by pruning some parameters of the diffusion model. Specifically, CR-Diff has two stages. It first performs block-wise pruning to selectively eliminate adverse weights. Then, a pruned output amplification is conducted to further purify the pruned predictions. Empirically, extensive experiments suggest that CR-Diff can improve perceptual fidelity and semantic coherence across various diffusion backbones and unseen resolutions, while largely preserving the performance at default resolutions. Additionally, CR-Diff supports prompt-specific refinement, enabling quality enhancement on demand.

## 一、核心问题与研究背景

- **问题定义**：当前主流的文本到图像扩散模型（如 Stable Diffusion 系列、SDXL 等）大多基于 UNet 架构，且训练分辨率固定（常见为 512×512 或 1024×1024）。这些模型在生成与训练分辨率不同的图像时，往往出现视觉伪影、语义对齐不良和结构失真。
- **研究动机**：该问题源于模型权重的“分辨率依赖性”（resolution-dependent parameters）。即在默认分辨率下表现良好的权重，在尺度变化时变得不稳定或甚至“有害”。传统解决方案如多分辨率微调或 DiT 架构虽有效，但成本高且结构复杂。
- **研究目标**：通过一种轻量化的**网络剪枝（network pruning）**策略，提高扩散模型的跨分辨率生成一致性，而不改变原有网络架构或重新训练。

## 二、方法论与技术流程

### 2.1 核心思想

提出 **CR-Diff（Cross-Resolution Diffusion Model via Network Pruning）**，通过有针对性的剪枝策略“净化”模型权重，保留稳定的参数、去除引入结构失稳的部分。剪枝不仅用于模型压缩，更被重新定义为提升生成质量的手段。

### 2.2 两阶段框架

1. **阶段一：块级剪枝（Block-Wise Pruning）**
   - 针对 UNet 的三种结构模块 —— 下采样块（Down）、中间块（Middle）、上采样块（Up） —— 采用不同的剪枝比例。
   - 使用**基于权重幅度的标准（Magnitude-based Criterion）**选择保留权重，并通过**模拟退火搜索算法（Simulated Annealing, SA）**确定最佳剪枝比例配置。
   - 每个分块的剪枝比例记为 `r = {r_down, r_mid, r_up}`，优化目标为在 ImageReward 等生成指标上取得最大值。

2. **阶段二：剪枝输出放大（Pruned Output Amplification, POA）**
   - 在每个去噪步骤 t，模型输出由剪枝模型 `z_Pt` 和原始稠密模型 `z_Dt` 组合：
     \[
     z_t = k z_Pt + (1 - k) z_Dt
     \]
   - 通过设置放大系数 `k > 1`，放大剪枝带来的“有益方向”，抑制稠密模型中残余的伪影信号。
   - 实验证明最佳系数为 `k = 1.5`。

3. **扩展功能：提示词特定优化（Prompt-Specific Refinement）**
   - 对单个文本提示进行微调搜索最优剪枝配置，实现针对性生成质量提升。
