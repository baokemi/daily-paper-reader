## 摘要
目标指称引导的扫描路径预测（Object Referring-guided Scanpath Prediction, ORSP）旨在根据描述目标对象的语言描述，预测人在视觉场景中搜索特定目标对象时的注意扫描路径。多模态信息融合是 ORSP 的关键环节。为此，我们提出了一种新模型 ScanVLA，首先利用视觉语言模型（Vision-Language Model, VLM）从输入图像和指称表达中提取并融合本身对齐的视觉和语言特征表示。接着，为增强 ScanVLA 对细粒度位置信息的感知能力，我们不仅提出了新颖的历史增强扫描路径解码器（History Enhanced Scanpath Decoder, HESD），将历史凝视点的位置信息直接作为输入，以帮助预测当前凝视点更合理的位置，还采用冻结的分割 LoRA 作为辅助组件，以更准确地定位被指称的对象，从而在不产生额外大量计算和时间成本的情况下提升扫描路径预测任务。大量实验结果表明，ScanVLA 在目标指称场景下能够显著优于现有的扫描路径预测方法。

## Abstract
Object Referring-guided Scanpath Prediction (ORSP) aims to predict the human attention scanpath when they search for a specific target object in a visual scene according to a linguistic description describing the object. Multimodal information fusion is a key point of ORSP. Therefore, we propose a novel model, ScanVLA, to first exploit a Vision-Language Model (VLM) to extract and fuse inherently aligned visual and linguistic feature representations from the input image and referring expression. Next, to enhance the ScanVLA's perception of fine-grained positional information, we not only propose a novel History Enhanced Scanpath Decoder (HESD) that directly takes historical fixations' position information as input to help predict a more reasonable position for the current fixation, but also adopt a frozen Segmentation LoRA as an auxiliary component to help localize the referred object more precisely, which improves the scanpath prediction task without incurring additional large computational and time costs. Extensive experimental results demonstrate that ScanVLA can significantly outperform existing scanpath prediction methods under object referring.

## 1. 核心问题与研究背景

- **研究主题**：论文聚焦于“目标指称引导的扫描路径预测”（Object Referring-Guided Scanpath Prediction, ORSP）任务。  
- **问题定义**：在给定图像与对目标对象的自然语言描述下，预测人在寻找目标对象时的**视觉凝视路径（scanpath）**。  
- **研究动机**：
  - 传统的扫描路径预测模型（如 ART）在视觉与语言特征融合上存在对齐困难，且需要额外的辅助任务（如目标分类、定位）预训练，极大地增加计算负担。
  - 现代视觉语言模型（VLM）具备良好的视觉语义协同能力，可天然解决模态对齐问题，但其弱点在于对空间分辨率和位置信息的细粒度感知不足。
- **研究意义**：提升计算机模拟人类视觉注意机制的能力，可为 **人机交互、自动驾驶、机器人视觉、VR/AR 等领域**提供更加符合人类认知规律的模型基础。

## 2. 方法论

### 2.1 整体架构：ScanVLA

- **核心思想**：提出一个基于视觉语言模型的统一框架 **ScanVLA**（Vision-Language-Action 模型），实现从图像与语言输入到扫描路径输出的端到端预测。
- **流程概述**：
  1. **输入阶段**：图像 \( I \) 与指称表达 \( R = \{[BOT], w_1, w_2, ..., w_L, [EOT]\} \)；
  2. **VLM 模块（特征融合）**：
     - 使用预训练模型如 InternVL 2.5 (1B) 和 Qwen3-VL (2B) 提取视觉-语言联合隐状态；
     - 引入两种 **LoRA 模块**：
       - **Segmentation LoRA**（冻结）：由 Sa2VA 迁移而来，增强模型的分割与对象定位能力；
       - **Scanpath LoRA**（可训练）：适配 ORSP 任务，学习凝视序列特征；
     - 利用 **因果掩码（causal mask）** 保证仅利用已见词预测当前凝视路径，符合人类逐词听觉特征。
  3. **HESD（History Enhanced Scanpath Decoder）**：
     - 以历史凝视点的 \((x, y)\) 坐标及顺序信息为输入；
     - 使用单向 GRU 提取全局历史特征，再与文本嵌入融合；
     - 使用 **Multi-Fixation Predictor**（多个两层 MLP）预测每一步凝视点的坐标与有效性概率；
     - 动态决定每个词的凝视数量：当有效性 \( V < 0.5 \) 时终止当前词的凝视。
  4. **损失函数**：
     - 总损失 \( L_{total} = L_{txt} + L_{xy} + L_{token} \)
       - \( L_{txt}\)：文本自回归交叉熵损失；
       - \( L_{xy}\)：基于L1的拟合凝视位置误差；
       - \( L_{token}\)：基于焦点损失（Focal Loss）的有效性预测损失。

## 3. 实验设计

- **使用数据集**：
  - **RefCOCO-Gaze**：目前唯一适用于 ORSP 的公开数据集，包含 2,094 幅 COCO 图像与 19,738 条人工凝视路径。
- **评估指标**：
  - 扫描路径相似度（SS、SS_pack）、路径编辑距离（FED、FED_pack），以及注视显著性指标（CC_pack、NSS_pack）。
  - 指标越高（SS、CC、NSS）表示预测路径越接近真实人类行为，FED 越低表示路径更准确。
- **对比基线**：
  - Human（人工平均表现上限）；
  - Random（随机路径）；
  - OFA、Chen et al.、Gazeformer-ref、Gazeformer-cat；
  - **ART**（当前 SOTA 模型）。
- **结果概述**：
  - ScanVLA（1B 和 2B）在所有指标上显著超越 ART；
  - 在部分指标上甚至超过人类平均值，说明生成路径更加一致、去除了主观差异噪声。
