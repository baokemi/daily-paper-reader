## 摘要
我们提出了 MMCORE，这是一种用于多模态图像生成和编辑的统一框架。MMCORE 利用预训练的视觉-语言模型（Vision-Language Model，VLM），通过可学习的查询 token 来预测语义视觉嵌入，这些嵌入随后作为扩散模型的条件信号。该简化设计有效地将 VLM 丰富的理解和推理能力转移到视觉生成过程中。通过避免在自回归与扩散模型之间进行深度融合或从零开始训练，MMCORE 显著降低了计算开销，同时保持高保真度的合成效果。MMCORE 无缝整合文本到图像合成与交错的图像生成，在空间推理与视觉定位等复杂场景中展现出强大的多模态理解能力。全面的评估表明，MMCORE 在广泛的文本到图像以及单/多图像编辑基准上始终超越最新的基线方法。

## Abstract
We present MMCORE, a unified framework designed for multimodal image generation and editing. MMCORE leverages a pre-trained Vision-Language Model (VLM) to predict semantic visual embeddings via learnable query tokens, which subsequently serve as conditioning signals for a diffusion model. This streamlined design effectively transfers the rich understanding and reasoning capabilities of VLMs into the visual generation process. By obviating the need for deep fusion between autoregressive and diffusion models or training from scratch, MMCORE significantly reduces computational overhead while maintaining high-fidelity synthesis.   MMCORE seamlessly integrates text-to-image synthesis with interleaved image generation, demonstrating robust multimodal comprehension in complex scenarios such as spatial reasoning and visual grounding. Comprehensive evaluations indicate that MMCORE consistently outperforms state-of-the-art baselines across a broad spectrum of text-to-image and single/multi-image editing benchmarks.

- **问题背景**：  
  当前多模态生成（文本-图像一体化生成）主要由两类模型主导：  
  1. **自回归（Autoregressive, AR）模型**：擅长语义理解与逻辑推理，但视觉生成质量较弱。

## 二、方法论：MMCORE 框架设计与技术要点

### 1. 总体思路

MMCORE（**MultiModal Connection with Representation Aligned Latent Embeddings**）旨在：
- 利用 **预训练视觉语言模型（VLM）** 生成语义视觉潜在嵌入（visual latent embeddings）；
- 通过 **可学习查询 token（learnable query tokens）** 对齐这些潜变量；
- 将其作为 **扩散模型的条件输入**；
- 实现无需端到端联合训练的多模态理解与高保真视觉生成。

### 2. 模型结构概述

- **底层：MLLM 模块（Multimodal Large Language Model）**
  - 接收文本/图像输入及可学习的查询 token；
  - 输出语义视觉潜在嵌入；
  - 采用多阶段训练（fine-tuning + representation alignment）。
  
- **上层：扩散生成头（Diffusion Head, 基于 MMDiT）**
  - 利用得到的视觉潜在嵌入及文本嵌入进行图像生成；
  - 采用“block-causal attention”掩码机制控制交错生成；
  - 支持多参考（multi-image）和交替编辑任务。

### 3. 关键技术细节

- **(i) 联合微调 (Joint Fine-tuning)**  
  对 MLLM 主干进行全量微调，而非冻结权重，仅使用轻量适配器。保证视觉表征能力与语义一致。

- **(ii) 语义对齐蒸馏 (Semantic Visual Alignment via Distillation)**  
  引入冻结 ViT 或 SigLIP 等视觉编码器作为教师模型，在生成的视觉潜在与目标视觉特征间施加**余弦相似损失（公式(1)**）。  
  \[
  L_{\text{vis}} = \frac{1}{N}\sum_i (1 - \cos(F_\theta(Q_i), v_i))
  \]  
  最终总损失：  
  \[
  L_{\text{MLLM}} = \lambda_t L_{\text{llm}} + \lambda_a L_{\text{vis}}
  \]

- **(iii) 双通道调节 (Dual-Pathway Conditioning)**  
  同时保留：
  - 可学习视觉 query token（捕获全局语义与跨模态语义对齐）；
  - 原始多模态文本序列（保留细节指令与约束）。
  该设计解决了单一固定查询预算 N 带来的瓶颈。

- **(iv) 独立嵌入 Dropout 训练策略**  
  在扩散阶段，对文本与视觉嵌入使用不同 dropout 率：  
  - 初期提升视觉嵌入权重（提高视觉捕捉能力）；  
  - 后期减小 dropout 平衡语义与结构信号。

- **(v) 训练流程**
  1. 阶段一：MLLM 两阶段优化：LLM 理解损失 + 视觉蒸馏损失；  
  2. 阶段二：扩散模型在冻结的 MLLM 条件下继续预训练 (Continued Pretraining, CPT)；  
  3. 阶段三：监督微调 (SFT) 与 RLHF 对齐（人类偏好增强）。
