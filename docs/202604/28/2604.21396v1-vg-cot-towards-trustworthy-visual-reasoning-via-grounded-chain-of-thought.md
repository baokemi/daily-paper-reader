## 摘要
大型视觉语言模型（Large Vision-Language Models, LVLMs）的发展需要精确的局部区域推理，使模型的逻辑能够忠实地基于实际视觉证据进行。现有数据集由于需要大量人工标注而在可扩展性方面受到限制，并且多步推理与对应图像区域之间缺乏明确的对齐，从而制约了模型可信度的评估。为应对这些挑战，我们提出了视觉锚定思维链（Visual Grounding Chain-of-Thought, VG-CoT）数据集，该数据集通过全自动的三阶段流程，将每一步推理显式地与图像中的真实视觉证据相联系。该流程首先利用最先进的检测模型和OCR模型提取对象级和文本级的视觉证据，然后使用GPT-4o生成逐步的锚定推理，最后通过基于推理依据的开放集检测过程对锚定进行优化。此外，我们还提出了一个新的基准，用于从三个互补维度全面评估LVLMs的推理性能：推理依据质量、答案准确性和推理-答案一致性。我们通过对包括LLaVA-1.5和Qwen2-VL在内的代表性LVLMs进行实验，结果显示在大多数评估指标上均取得了一致性提升，验证了VG-CoT在保持数据集构建可扩展性和成本效益的同时，能够有效增强基于证据的可信推理。数据集和代码将在论文接收后公开发布，以促进后续研究。

## Abstract
The advancement of Large Vision-Language Models (LVLMs) requires precise local region-based reasoning that faithfully grounds the model's logic in actual visual evidence. However, existing datasets face limitations in scalability due to extensive manual annotation and lack of explicit alignment between multi-step reasoning and corresponding image regions, which constrains the evaluation of model trustworthiness. To address these challenges, we propose the Visual Grounding Chain-of-Thought (VG-CoT) dataset, which explicitly links each reasoning step to real visual evidence within the image through a fully automated three-stage pipeline. The pipeline first extracts object- and text-level visual evidence using state-of-the-art detection and OCR models, then generates step-by-step grounded reasoning with GPT-4o, and finally refines the grounding through a rationale-driven open-set detection process. In addition, we introduce a new benchmark that comprehensively evaluates LVLMs reasoning across three complementary dimensions: Rationale Quality, Answer Accuracy, and Reasoning-Answer Alignment. Experiments with representative LVLMs, including LLaVA-1.5 and Qwen2-VL, demonstrate consistent improvements on most evaluation metrics, confirming that VG-CoT effectively enhances trustworthy, evidence-based reasoning while maintaining scalable and cost-efficient dataset construction. The dataset and code will be released publicly upon acceptance to facilitate further research.

## 一、研究背景与问题动机

- **研究背景**：  
  随着大型视觉语言模型（Large Vision-Language Models, LVLMs）的发展，其多模态推理能力不断增强，但目前多停留在**图像整体理解层面**，缺乏针对**局部区域（局部对象、文本区域）**的精细化推理能力。  
  然而，真实世界任务（如自动驾驶、医疗影像、机器人视觉）往往要求模型在局部层面进行多步、逻辑一致的推理。

- **主要问题**：  
  现有多模态推理数据集和Benchmark存在以下显著局限：
  1. 严重依赖人工标注，**成本高且难以扩展**。  
  2. 缺乏推理过程与图像中**具体视觉证据（bounding boxes）**的显式对应。  
  3. 仅以“答案准确率”为考核，无法判断模型的推理逻辑是否可信、是否真正依据视觉信息。  

- **研究目标**：  
  作者希望**同时实现数据构建的自动化与推理过程的证据对齐（grounding）**，以提升LVLMs在解释性和信任度上的评估与训练能力。

## 二、方法论与技术框架

### 核心思想
提出**VG-CoT（Visual Grounding Chain-of-Thought）**数据集及其生成流程，通过全自动化三阶段管道，将每一步推理与视觉证据（对象、文本）精确绑定，从而得到“可验证逻辑链”。

### 三阶段自动化流程（如图 2 所示）
1. **阶段 1：多粒度视觉证据提取**
   - 使用 **YOLOvX** 对象检测模型提取对象级 bounding boxes。
   - 使用 **PaddleOCR** 提取图像中文字级 bounding boxes 及内容。
   - 若源数据集（如GQA）已有场景图（scene graph），则一并利用。

2. **阶段 2：生成视觉锚定的思维链（Visually-Grounded CoT Generation）**
   - 输入：图像、问题、标准答案与阶段1的视觉证据信息。
   - 执行者：**GPT-4o**（具备多模态理解与生成能力）。
   - 输出：逐步推理文本，并**显式嵌入坐标引用**（例如 `[0.086,0.637,0.250,0.909]`）。  
   - 提示模板要求模型：
     - 展示逐步推理；
     - 引用视觉证据信息；
     - 说明错误选项；
     - 指出何种视觉线索支持最终结论。

3. **阶段 3：推理驱动的锚定精修（Rationale-Driven Grounding Refinement）**
   - 使用 NLP 工具（如 nltk）提取推理文本中的对象名；
   - 调用 **Grounding DINO（open-set detection model）** 进行局部语义-视觉精修；
   - 将检测到的新证据重新映射至推理步骤，实现严密的“文字-图像位置”对齐。

> 整个流程完全自动化，无人工干预，兼顾可扩展性与一致性。

## 三、实验设计与评估设置

### 数据集来源与构造
- 覆盖三类任务场景，以提升模型跨域推理能力：
  1. **General VQA**（Visual7W，4,000条） — 对象辨识；
  2. **Relation VQA**（GQA，6,600条） — 关系推理；
  3. **Scene-text VQA**（TextVQA，3,226条） — 图文融合理解。
- 共计 **13,826 样本**：
  - 训练集 10,226，验证集 1,800，测试集 1,800。

### 评估基准设计
提出全新三维评价体系：
1. **推理依据质量 RQ（Rationale Quality）**
   - 子指标：视觉证据利用度、逻辑连贯性、推理完整性。
   - 每项评分范围：1–5。
2. **答案准确率 AA（Answer Accuracy）**
   - 最终答案的正确率。
3. **推理-答案一致性 RAA（Reasoning-Answer Alignment）**
   - 定义两个派生指标：
     - **Consistency Score**：高质量推理与正确答案匹配的比例；
     - **Faithful Score**：在高质量推理样本中，答案正确的比例。

### 对比模型
- **LLaVA-1.5（7B / 13B）**
- **Qwen2-VL-7B-Instruct**
- **Qwen2.5-VL-7B-Instruct**

### 实验指标
- 除上述推理与一致性指标外，还计算**视觉证据预测的 mAP（@0.5 / @0.75）** 用以验证模型的空间定位精度。

## 六、主要结论与发现

- **VG-CoT微调显著提升信任式视觉推理能力：**
  - LLaVA-1.5-7B：整体RQ从72.2 ↑至83.4，AA从48.7% ↑至62.5%。  
  - Qwen2.5-VL-Instruct：RQ从88.0 ↑至89.5，AA从68.5% ↑至73.6%。  
  - 说明经锚定思维链训练的模型，推理逻辑更连贯、答案更准确。
- **多任务泛化性**：  
  所有任务（General、Relation、Scene-text）均获得大幅提升，其中小模型增益最显著。
- **视觉证据预测分析**：
  - LLaVA-1.5-13B 在 mAP@0.5 达 48.7（较7B高约4点），但在 mAP@0.75 降低明显，说明模型可粗略对齐证据，但精确定位仍具挑战。
- **研究意义**：  
  VG-CoT提供了首个“证据级推理链”资源，为评测LVLMs的可信性和解释性提供了系统化框架。
