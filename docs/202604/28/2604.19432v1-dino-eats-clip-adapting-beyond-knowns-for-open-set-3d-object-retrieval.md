## 摘要
视觉基础模型通过高效地适配多视图图像，在开放集三维目标检索（3DOR）中展现出巨大潜力。以语义对齐的潜在空间为基础，以往研究通常将 CLIP 编码器进行适配以构建基于视图的三维描述符。尽管 CLIP 具有较强的泛化能力，但其缺乏细粒度特征促使我们探索更新的自监督编码器——DINO 的潜力。为此，我们提出了 DINO Eats CLIP（DEC），一个通过为未见类别合成数据进行正则化的动态多视图融合新框架。我们首先发现，在冻结的 DINO 主干上对视图特征进行简单的平均池化即可获得不错的性能。然而进一步的适配会导致对已知类别的平均视图模式严重过拟合。为解决这一问题，我们设计了分块与自适应模块（Chunking and Adapting Module, CAM），该模块将多视图图像分割为若干块并动态整合局部视图关系，从而生成比标准池化策略更鲁棒的特征。最后，我们提出虚拟特征合成模块（Virtual Feature Synthesis, VFS），以显式减轻模型对已知类别的偏差。VFS 利用 CLIP 广泛且已对齐的视觉-语言空间，为未见类别合成虚拟特征。通过让 DEC 接触这些虚拟特征，我们显著提升了其开放集区分能力。在标准开放集 3DOR 基准上的大量实验验证了该方法的卓越效能。

## Abstract
Vision foundation models have shown great promise for open-set 3D object retrieval (3DOR) through efficient adaptation to multi-view images. Leveraging semantically aligned latent space, previous work typically adapts the CLIP encoder to build view-based 3D descriptors. Despite CLIP's strong generalization ability, its lack of fine-grainedness prompted us to explore the potential of a more recent self-supervised encoder-DINO. To address this, we propose DINO Eats CLIP (DEC), a novel framework for dynamic multi-view integration that is regularized by synthesizing data for unseen classes. We first find that simply mean-pooling over view features from a frozen DINO backbone gives decent performance. Yet, further adaptation causes severe overfitting on average view patterns of known classes. To combat it, we then design a module named Chunking and Adapting Module (CAM). It segments multi-view images into chunks and dynamically integrates local view relations, yielding more robust features than the standard pooling strategy. Finally, we propose Virtual Feature Synthesis (VFS) module to mitigate bias towards known categories explicitly. Under the hood, VFS leverages CLIP's broad, pre-aligned vision-language space to synthesize virtual features for unseen classes. By exposing DEC to these virtual features, we greatly enhance its open-set discrimination capacity. Extensive experiments on standard open-set 3DOR benchmarks demonstrate its superior efficacy.

- **核心问题**：如何在开放集（open-set）环境下实现鲁棒的三维对象检索（3D Object Retrieval, 3DOR），即使测试类未在训练中出现，模型仍能正确匹配相似形状。

  - 现有开放集方法多依赖 **CLIP** 等视觉-语言模型，通过图像与文本语义对齐提升泛化性。

## 二、方法论：核心思想与技术细节

### 2.1 整体框架

- **框架名称**：DINO Eats CLIP（简称 DEC）  
- **总体思路**：结合 DINO 的细粒度自监督视觉特征与 CLIP 的语义泛化能力，以提升开放集下的三维检索性能。  
- 模型包含三个核心模块：  
  - **DINO 主干特征提取**  
  - **Chunking and Adapting Module（CAM）**  
  - **Virtual Feature Synthesis（VFS）**

### 2.2 主要流程

1. **多视图特征提取**  
   - 每个三维对象被渲染为多视图图像（24 张）。  
   - 通过冻结的 DINO 编码器提取每个视图的 `[CLS]` 特征向量 \( f_m = F^\circ(I_m) \)，形成特征序列 \([f_1, f_2, ..., f_M]\)。

2. **分块与自适应模块（CAM）**  
   - 将多视图特征划分为若干 **chunk（块）**，每块包含连续的视图特征，使用 1D 卷积、BN、ReLU 构成的 CBR 单元进行局部聚合。  
   - 再通过 **跨块池化** 捕获远程视图关系。  
   - 最后通过 **加权残差融合** 保存原始 DINO 特征与适配后的特征：
     \[
     g_\text{final} = \lambda \cdot f_\text{gap} + (1-\lambda) \cdot g_\text{adpt}
     \]
     其中 \(\lambda\) 控制预训练特征和适配特征的平衡。

3. **虚拟特征合成模块（VFS）**  
   - 利用 CLIP 的图像-文本对齐空间，将文本语义差 \((y_u - y_i)\) 投射到视觉空间。  
   - 对已知类视图特征 \(f_i^m\) 进行语义方向平移：
     \[
     v_u^m = \psi[f_i^m + \epsilon \cdot \text{LN}(y_u - y_i)]
     \]
     其中 \(\psi\) 为轻量 MLP，LN 为层归一化，\(\epsilon\) 可学习。  
   - 最终形成虚拟的未见类特征，用于训练阶段的正则化；该过程不增加推理成本。

4. **理论分析**  
   - 作者证明 CLIP 的视觉与文字空间在局部近似保持 **同构（isometric）语义关系**，因此从文本方向生成虚拟视觉特征在理论上可视为对未知类的广义采样。

5. **优化与损失**  
   - 采用 **multi-similarity loss** 进行度量学习，使相似类特征靠近、不同类特征远离。  
   - 推理阶段使用 CAM 的输出作为最终三维描述符，无需文本或虚拟特征。

## 六、主要结论与发现

1. 简单的多视图 **DINO 平均池化基线** 已优于所有 CLIP-based 方法，证明 DINO 的细粒度自监督特征更适合 3D 表征。  
2. **CAM 模块** 可显著提升模型对局部几何信息的整合能力，减少过拟合。  
3. **VFS 模块** 借助 CLIP 的语义指导生成虚拟特征，有效提升模型在未知类别上的辨识性能。  
4. 整体框架在四个开放集基准上刷新 SOTA，例如在 OS-MN40-core 上的 mAP 提升超过 +5%。  
5. 推理阶段只依赖单模态视觉输入，无需文本或多模态信息，效率更高。

## 七、优点与创新亮点

- 利用 **DINO 的细粒度特征提取能力** 解决 CLIP 的语义过度全局化问题。  
- 设计 **分块自适应聚合（CAM）** 模块，以分而治之方式建模多视图间的局部关联。  
- 提出 **CLIP 引导的虚拟特征正则化（VFS）**，实现对未知类别的“伪曝光”。  
- 理论上证明了文本空间转移到视觉空间的合理性（语义同构性）。  
- 推理端轻量化，仅需视觉特征，具备较高实用性。  
- 实验覆盖多个规模与难度梯度的数据集，验证方法的泛化与稳定性。
