## 摘要
变化视觉问答（Change VQA）旨在回答关于双时相遥感（RS）图像语义变化的自然语言问题。尽管视觉语言模型（VLM）近年来已被用于时序遥感图像理解，但在现代多模态模型背景下，对 Change VQA 的探索仍然不足。本文在统一的低秩适配（LoRA）设置下，基于最新的 Qwen 模型重新评估了 CDVQA 基准。我们比较了 Qwen3-VL（采用结构化视觉语言管线、具备多层视觉条件控制和全注意力解码器）与 Qwen3.5（原生多模态模型，结合单阶段对齐与混合解码骨干网络）之间的表现。官方 CDVQA 测试集上的实验结果表明，最新的 VLM 相比早期的专用基线方法有显著改进。此外，结果显示性能并不随模型规模单调提升，且原生多模态模型在此任务上优于结构化视觉语言管线。这些发现表明，对于遥感影像中的语言驱动语义变化推理，紧密集成的多模态骨干网络比模型规模或显式多层视觉条件控制对性能的提升更为关键。

## Abstract
Change visual question answering (Change VQA) addresses the problem of answering natural-language questions about semantic changes between bi-temporal remote sensing (RS) images. Although vision-language models (VLMs) have recently been studied for temporal RS image understanding, Change VQA remains underexplored in the context of modern multimodal models. In this letter, we revisit the CDVQA benchmark using recent Qwen models under a unified low-rank adaptation (LoRA) setting. We compare Qwen3-VL, which follows a structured vision-language pipeline with multi-depth visual conditioning and a full-attention decoder, with Qwen3.5, a native multimodal model that combines a single-stage alignment with a hybrid decoder backbone. Experimental results on the official CDVQA test splits show that recent VLMs improve over earlier specialized baselines. They further show that performance does not scale monotonically with model size, and that native multimodal models are more effective than structured vision-language pipelines for this task. These findings indicate that tightly integrated multimodal backbones contribute more to performance than scale or explicit multi-depth visual conditioning for language-driven semantic change reasoning in RS imagery.

## 二、方法论

### 1. 核心思想
- 以 **Qwen 家族多模态模型** 为研究对象，比较两种多模态融合策略：
  - **Qwen3-VL（结构化管线）**：采用分级视觉特征融合，利用 PatchMerger + 多层 DeepStack 视觉条件注入；
  - **Qwen3.5（原生多模态）**：单阶段视觉-语言对齐，后续通过 GatedDeltaNet + 全注意力混合解码器实现原生信息交互。
- 目标不是设计新网络，而是验证“多模态融合深度 vs. 模型规模”的性能关系。

### 2. 任务定义
- 输入：两幅对齐的双时相图像 \( I^{(1)}, I^{(2)} \) 与自然语言问题 \( q \)；
- 输出：对应的答案 \( a \)，其预测通过最大化条件概率：
  \[
  y^* = \arg\max_{y \in Y} p_\theta(y | I^{(1)}, I^{(2)}, q)
  \]
- 模型以**自回归生成**方式输出答案序列。

### 3. 参数高效适配
- 采用 **LoRA（Low-Rank Adaptation）** 技术微调模型。
  - 仅对解码器的 Q/K/V/O 投影层注入可训练低秩矩阵，保持主干参数冻结；
  - 超参数设置：LoRA rank = 16，缩放系数 α = 32；
  - 各模型可训练参数比例仅 0.3%–0.8%。

### 4. 模型对比结构
- **Qwen3-VL**：  
  - 视觉编码器（ViT）→ PatchMerger 对齐 → DeepStack ×3 多层注入 → 全注意力解码器；
- **Qwen3.5**：  
  - 相同视觉前端 → 单次 PatchMerger → 混合解码器（GatedDeltaNet + Full-Attn）；
- 差异核心：后续交互机制是否显式多层视觉条件注入，或采用紧密一体化解码。

### 2. 对比方法
- **横向对比**：
  - Qwen3-VL 与 Qwen3.5 的多个规模版本（0.8B–9B）；

- 性能评价：按问题类型计算准确率，并汇总平均准确率（AA）与总体准确率（OA）。

## 六、主要结论与发现

1. **现代通用VLM能够有效处理变化问答任务**，无需专门的变化检测模块；
2. **原生多模态（Qwen3.5）优于结构化管线（Qwen3-VL）**，表明紧密融合的信息交互比多层视觉条件更有效；
3. **模型规模不与性能单调相关**：增大模型（如8B或9B）未带来明显收益；
4. **最佳效能平衡模型为 Qwen3.5-2B**：在 Test1 OA=74.74%、Test2 OA=70.94%，超过此前最优方法 VisTA；
5. **主要误差类型**：
   - 细微变化下的假阳性；
   - 最小变化类别的混淆；
   - 定量变化比区间判断误差。
