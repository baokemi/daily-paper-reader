## 摘要
尽管近年来取得了显著进展，视觉-语言编码器仍面临两个核心限制：（1）语言与密集视觉特征之间的对齐较弱，这对开放词汇语义分割等任务造成不利影响；（2）细粒度视觉表示的标记数量庞大，限制了对长视频的可扩展性。本研究针对这两个问题提出了解决方案。我们提出了 T-REN（Text-aligned Region Encoder Network，文本对齐区域编码网络），这是一种高效编码器，可将视觉数据映射为一组紧凑的、与文本对齐的区域级表示（或区域标记）。T-REN 通过在冻结的视觉主干上添加一个轻量级网络实现：该网络被训练用于在每个语义区域内将 patch 级表示汇聚为区域标记，并将其与区域级文本标注对齐。与视觉-语言主干相比，仅增加约 3.7% 的参数量，这一设计便显著增强了密集的跨模态理解，同时将标记数量减少了几个数量级。具体而言，T-REN 在 ADE20K 开放词汇分割任务上提高了 +5.9 的 mIoU，在 COCO 对象级文本-图像检索任务上提高了 +18.4% 的召回率，在 Ego4D 视频对象定位上提高了 +15.6% 的召回率，在 VSPW 视频场景解析上提高了 +17.6% 的 mIoU。同时，与基于 patch 的视觉-语言主干相比，其图像标记数减少超过 24 倍，视频标记数减少超过 187 倍。代码与模型可在 https://github.com/savya08/T-REN 获取。

## Abstract
Despite recent progress, vision-language encoders struggle with two core limitations: (1) weak alignment between language and dense vision features, which hurts tasks like open-vocabulary semantic segmentation; and (2) high token counts for fine-grained visual representations, which limits scalability to long videos. This work addresses both limitations. We propose T-REN (Text-aligned Region Encoder Network), an efficient encoder that maps visual data to a compact set of text-aligned region-level representations (or region tokens). T-REN achieves this through a lightweight network added on top of a frozen vision backbone, trained to pool patch-level representations within each semantic region into region tokens and align them with region-level text annotations. With only 3.7% additional parameters compared to the vision-language backbone, this design yields substantially stronger dense cross-modal understanding while reducing the token count by orders of magnitude. Specifically, T-REN delivers +5.9 mIoU on ADE20K open-vocabulary segmentation, +18.4% recall on COCO object-level text-image retrieval, +15.6% recall on Ego4D video object localization, and +17.6% mIoU on VSPW video scene parsing, all while reducing token counts by more than 24x for images and 187x for videos compared to the patch-based vision-language backbone. The code and model are available at https://github.com/savya08/T-REN.

## 一、研究问题与背景

- **核心问题**：当前视觉-语言编码器（如 CLIP、DINOv3-based dino.txt）在密集视觉理解任务中受到两大核心限制：  
  1. **跨模态对齐不足**：语言与视觉特征的对应关系粗糙，导致开放词汇分割、检索、定位任务性能低下。  
  2. **Token 规模过大**：现有模型以 patch 为单位进行视觉表征，每幅图片可能产生数千个 token，难以扩展到长视频或大规模场景。

- **研究动机**：  
  希望找到一种在保持语义精度的同时，显著降低视觉 token 数量、增强视觉与语言之间细粒度对齐的结构化编码方式。

- **整体目标**：构建一种高效且语义可解释的区域级视觉表示机制，使视觉语言模型在密集任务上更强、更可扩展。

## 二、方法论：T-REN 模型设计

### 核心思想

- 提出 **T-REN（Text-aligned Region Encoder Network）**：在冻结的视觉主干（DINOv3 ViT-L）上添加一个轻量模块，将 patch 特征聚合为语义一致的**区域标记（region tokens）**，并与文本特征对齐。
- 仅增加约 **3.7% 参数量**，即可显著提升密集跨模态对齐并减少 token 数量。

### 模型结构与技术细节

1. **点提示与查询生成**：  
   - 在图像上布置一个二维网格的点提示（point prompts）。  
   - 每个点通过学习得到 **k=3 个查询 embedding**，以捕获对象的不同语义层（整体与局部部分）。

2. **跨注意聚合（Cross-Attention Pooling）**：  
   - 两层解码器结构，利用跨注意力将查询与 patch token 交互，聚合空间上相似的 patch 特征。  
   - 每个查询产生一个低分辨率的语义掩码（region mask），用于汇聚对应区域的信息。

3. **Token 合并机制**：  
   - 通过余弦相似度（τtoken=0.975）或 mask IoU（τmask=0.8）判断语义相似区域并进行平均池化合并，减少冗余 token。  
   - 合并后得到自适应数量的区域标记 M（随图像复杂度变化）。

4. **文本对齐投影**：  
   - 使用两层 MLP（1024→2048→1024）将视觉区域 token 投影到文本嵌入空间，以实现区域级语义对齐。

5. **视频扩展：跨帧聚合（Temporal Merging）**：  
   - 将相似区域 token 在连续帧中聚合为“轨迹 token（track token）”，通过余弦相似度阈值 τtrack=0.65 确定对应关系，实现视频中对象长期跟踪与压缩。

### 训练目标与公式说明（文字描述）

- **对比学习（Contrastive Loss）**：在视觉与文本空间同时施加区域级对比损失，促使同一语义区域的 token 聚类，不同区域分离。
- **蒸馏损失（Distillation Loss）**：保证新生成的区域 token 与冻结主干的特征空间保持一致，避免漂移。
- **注意力监督（Attention Supervision）**：强制 cross-attention mask 与真值区域掩码匹配，加速收敛。
- **Hungarian 匹配**：用于一对多区域预测的标签匹配，使训练具有排列不变性。
