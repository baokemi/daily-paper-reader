## 摘要
我们提出了 LLaDA2.0-Uni，这是一种统一的离散扩散大型语言模型（dLLM），在原生集成的框架中支持多模态理解与生成。其架构结合了完全语义化的离散分词器、基于 MoE 的 dLLM 主干以及扩散解码器。通过 SigLIP-VQ 将连续视觉输入离散化，模型在主干内部实现了针对文本和视觉输入的块级掩码扩散，而解码器则将视觉 token 重建为高保真图像。推理效率通过主干中的前缀感知优化及解码器中的少步蒸馏得到进一步提升。依托精心整理的大规模数据和定制的多阶段训练管线，LLaDA2.0-Uni 在多模态理解方面可与专用的 VLM 相媲美，并在图像生成与编辑任务中展现出强劲表现。其原生支持交错生成与推理，为下一代统一基础模型建立了一个有前景且可扩展的范式。代码与模型详见：https://github.com/inclusionAI/LLaDA2.0-Uni。

## Abstract
We present LLaDA2.0-Uni, a unified discrete diffusion large language model (dLLM) that supports multimodal understanding and generation within a natively integrated framework. Its architecture combines a fully semantic discrete tokenizer, a MoE-based dLLM backbone, and a diffusion decoder. By discretizing continuous visual inputs via SigLIP-VQ, the model enables block-level masked diffusion for both text and vision inputs within the backbone, while the decoder reconstructs visual tokens into high-fidelity images. Inference efficiency is enhanced beyond parallel decoding through prefix-aware optimizations in the backbone and few-step distillation in the decoder. Supported by carefully curated large-scale data and a tailored multi-stage training pipeline, LLaDA2.0-Uni matches specialized VLMs in multimodal understanding while delivering strong performance in image generation and editing. Its native support for interleaved generation and reasoning establishes a promising and scalable paradigm for next-generation unified foundation models. Codes and models are available at https://github.com/inclusionAI/LLaDA2.0-Uni.

## 二、方法论与核心技术

### 1. 总体架构
LLaDA2.0-Uni 是一个 **统一离散扩散大语言模型（dLLM）**，由三大模块组成：
- **SigLIP-VQ 离散化视觉分词器**：将图像转化为语义化离散 token；
- **MoE 扩散语言模型主干（16B 参数）**：统一处理文本与视觉 token；
- **扩散解码器**：将视觉 token 重建为高保真图像。

### 2. 核心思想
- 使用 **完全语义离散化的表示**（非像素重建），以实现文本和图像的统一 token 格式；
- 在主干模型中引入 **块级掩码扩散（Block Diffusion）**，强化上下文建模和并行预测；
- 通过 **Mixture-of-Experts (MoE)** 架构实现跨模态容量的动态分配，不需独立视觉/文本模块；
- 引入 **前缀优化（SPRINT）** 与 **少步蒸馏（8步CFG-free生成）** 提升推理效率。

### 3. 关键技术细节（文字说明）
- **SigLIP-VQ Tokenizer**：
  - 基于 SigLIP2-ViT 提取视觉特征；
  - 采用语义对齐的向量量化（codebook 容量 16384，维度 2048）；
  - 输出离散 token，其语义直接与语言模型共享。
- **扩散语言模型（BDLM）损失函数**：  
  建模块级 mask 区域的预测任务。每个块在扩散步骤中进行去噪采样：
  ```
  L_BDLM = - E_t,x [ Σ 块内掩码位置 log Pθ(x_masked | x_clean, x_t) ]
  ```
- **负载均衡策略**：通过无辅助损失的MoE调度，避免专家塌陷；
- **Complementary Masking**：对每个样本生成两条互补mask序列，提升数据利用率。

### 4. 推理加速 (SPRINT)
- **Sparse Prefix Retention**：根据 token 置信度和关键向量范数裁剪注意力前缀，减少计算；
- **Non-uniform Token Unmasking**：基于置信度动态调整去噪步数，平均提升推理速度1.6×。
