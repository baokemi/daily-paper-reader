## 摘要
源无域自适应（Source-Free Domain Adaptation，SFDA）旨在使一个在有监督源域上预训练的源模型能够适应目标域，而仅能访问未标注的目标训练数据。传统方法依赖伪标签和/或辅助监督，因而不可避免地易出错。为缓解这一局限，本文首次探索利用现成的视觉-语言（Vision-Language，ViL）多模态模型（例如 CLIP），其具有丰富但异质的知识潜力。我们发现，直接以零样本方式将 ViL 模型应用于目标域效果不佳，因为该模型并未针对这一特定任务进行专门化，而只是泛化的。为使其具备任务特异性，我们提出了一种新颖的 DIFO++ 方法。具体而言，DIFO++ 在适应过程中交替执行两个步骤：（i）通过最大化与目标模型的互信息，以提示学习的方式定制 ViL 模型；（ii）将该定制化的 ViL 模型知识蒸馏至目标模型，聚焦于间隙区域的减少。在渐进式知识自适应过程中，我们首先识别并关注间隙区域——其中包含纠缠且类别模糊的特征，因为它通常捕获更丰富的任务特定语义。随后，通过融合目标模型与 ViL 模型的预测，并借助记忆机制生成可靠的伪标签。最后，间隙区域的缩减由类别注意力与预测一致性引导以实现语义对齐，并辅以参考熵最小化来抑制不确定性。大量实验表明，DIFO++ 显著优于现有的最新方法。我们的代码和数据可在 https://github.com/tntek/DIFO-Plus 获得。

## Abstract
Source-Free Domain Adaptation (SFDA) seeks to adapt a source model, which is pre-trained on a supervised source domain, for a target domain, with only access to unlabeled target training data. Relying on pseudo labeling and/or auxiliary supervision, conventional methods are inevitably error-prone. To mitigate this limitation, in this work we for the first time explore the potentials of off-the-shelf vision-language (ViL) multimodal models (e.g., CLIP) with rich whilst heterogeneous knowledge. We find that directly applying the ViL model to the target domain in a zero-shot fashion is unsatisfactory, as it is not specialized for this particular task but largely generic. To make it task-specific, we propose a novel DIFO++ approach. Specifically, DIFO++ alternates between two steps during adaptation: (i) Customizing the ViL model by maximizing the mutual information with the target model in a prompt learning manner, (ii) Distilling the knowledge of this customized ViL model to the target model, centering on gap region reduction. During progressive knowledge adaptation, we first identify and focus on the gap region, where enclosed features are entangled and class-ambiguous, as it often captures richer task-specific semantics. Reliable pseudo-labels are then generated by fusing predictions from the target and ViL models, supported by a memory mechanism. Finally, gap region reduction is guided by category attention and predictive consistency for semantic alignment, complemented by referenced entropy minimization to suppress uncertainty. Extensive experiments show that DIFO++ significantly outperforms the state-of-the-art alternatives. Our code and data are available at https://github.com/tntek/DIFO-Plus.

## 二、方法论：DIFO++ 框架

### 2.1 核心思想
- 将 SFDA 与 ViL 模型结合，提出 **DIFO++（Distilling multimodal Foundation model, Plus）**。
- 整体流程为“**定制化—蒸馏式交替优化**”：  
  1. 利用互信息最大化实现 ViL 模型的**任务特定提示学习（Prompt Learning）**。  
  2. 将定制ViL模型的知识**蒸馏至目标模型**，重点关注“间隙区域”（gap region）的语义模糊问题。

---

### 2.2 模型总体结构
DIFO++ 包含两个阶段、三个关键模块（见文中 Fig. 2）：

#### 阶段 I：ViL 模型定制（Customization）
- ViL 模型（如CLIP）被冻结，只学习**类别特定提示向量**。
- 采用目标模型预测与ViL预测之间的**互信息最大化**作为优化目标：
  \[
  L_{S-I} = - \min_v \, \mathbb{E}_{x_i \in X_t} I(\theta_t(x_i), \theta_v(x_i, v))
  \]
  这种设计避免了KL散度中对单一预测分布的偏倚。

#### 阶段 II：间隙区域驱动的知识适应
1. **间隙区域发现**  
   - 提出新指标——**参考熵（Referenced Entropy）**，定义为：
     \[
     U(x_i) = H(\theta_t(x_i)) - \rho^k_i
     \]
     其中，\(\rho^k_i\) 为源模型初始熵的滑动平均参考值，用于衡量适应过程中的不确定性变化。
   - 按阈值 \(m_k\) 逐步识别高不确定性样本，形成“gap region”。

2. **间隙区域伪标签融合**  
   - 引入**记忆库机制（memory bank）**保存历史预测：
     \[
     \bar{p_i} = \tau p'_i + (1-\tau) p''_i
     \]
     其中 \(\tau\) 遵循指数分布，用以平衡目标模型 \(p'\) 与 ViL 模型 \(p''\) 的指导强度。

3. **间隙区域缩减（Gap Reduction）**
   - **类别注意力校正（L_cac）**：聚焦伪标签中最可能的前N类别，引导目标模型的注意力；
   - **预测一致性（L_pc）**：保持 ViL 与目标模型预测互信息最大化；
   - **区域压缩（L_rc）**：最小化参考熵，实现高低不确定样本边界的渐进收敛。  
     联合目标函数为：
     \[
     L_{S-II} = L_{pc} + \beta L_{cac} + \eta L_{rc}
     \]

#### 总体算法流程（Algorithm 1 简述）
1. 初始化目标模型与基础CLIP；
2. 每轮训练执行：
   - 更新ViL预测并优化提示（stage I）；
   - 基于参考熵识别gap区域、融合伪标签、执行知识蒸馏（stage II）。

## 八、不足与局限

- **模型访问要求高**：  
  ViL模型需可访问内部参数（提示学习阶段），**不适用于黑盒/云API场景**。
- **无防遗忘机制**：  
  连续多域学习下易出现灾难性遗忘效应。
- **在线性有限**：  
  参考熵依赖历史分布信息，不适应实时（Test-Time Adaptation）场景。
- **领域覆盖仍有限**：  
  主要在图像识别任务中验证，未实测检测、分割等复杂任务。
