## 摘要
尽管由可验证奖励的强化学习（RLVR）推动的大型推理模型（Large Reasoning Models，LRMs）取得了显著进展，但这一范式在监督极其昂贵或无法获得的专业或新颖领域中存在根本限制，这对测试时适应提出了关键挑战。虽然现有的测试时方法提供了一种潜在的解决方案，但它们受限于从静态查询集学习，容易过拟合于文本模式。为弥补这一缺陷，我们提出了测试时变分合成（Test-Time Variational Synthesis，TTVS），一个新颖的框架，使LRMs能够通过从无标签测试查询动态增强训练流而自我演化。TTVS包含两个协同模块：（1）在线变分合成，将静态测试查询转换为动态的、多样的语义等价变体流，促使模型学习潜在问题逻辑而非表面模式；（2）测试时混合探索，在合成变体之间平衡准确性驱动的开发与一致性驱动的探索。大量实验表明，TTVS在八种模型架构上均取得了优异性能。值得注意的是，仅使用无标签的测试时数据，TTVS不仅超过了其他测试时适应方法，而且超越了基于大量高质量标注数据训练的最先进的有监督强化学习方法。

## Abstract
Despite significant advances in Large Reasoning Models (LRMs) driven by reinforcement learning with verifiable rewards (RLVR), this paradigm is fundamentally limited in specialized or novel domains where such supervision is prohibitively expensive or unavailable, posing a key challenge for test-time adaptation. While existing test-time methods offer a potential solution, they are constrained by learning from static query sets, risking overfitting to textual patterns. To address this gap, we introduce Test-Time Variational Synthesis (TTVS), a novel framework that enables LRMs to self-evolve by dynamically augmenting the training stream from unlabeled test queries. TTVS comprises two synergistic modules: (1) Online Variational Synthesis, which transforms static test queries into a dynamic stream of diverse, semantically-equivalent variations, enforcing the model to learn underlying problem logic rather than superficial patterns; (2) Test-time Hybrid Exploration, which balances accuracy-driven exploitation with consistency-driven exploration across synthetic variants. Extensive experiments show TTVS yields superior performance across eight model architectures. Notably, using only unlabeled test-time data, TTVS not only surpasses other test-time adaptation methods but also outperforms state-of-the-art supervised RL-based techniques trained on vast, high-quality labeled data.

## 二、方法论：Test-Time Variational Synthesis (TTVS)

### 1. 总体思路
TTVS 通过将静态测试数据转化为动态“训练流”，让模型在推理过程中**自生成语义等价的变体问题**并用作训练数据。  
核心由两大模块组成：

1. **在线变分合成（Online Variational Synthesis）**
2. **测试时混合探索（Test-Time Hybrid Exploration）**

两者协同实现“自我样本生成 + 自我优化”，平衡准确性与一致性。

---

### 2. 模块一：在线变分合成（Online Variational Synthesis）

- **关键目标**：把原始测试查询 *q* 转化为多个语义等价但表达不同的变体 *{q′₁, q′₂,…}*。
- **步骤流程**：
  1. **伪标签生成**：通过多数投票（majority vote）获得每个测试样本的伪答案 *y\*_q* ；
  2. **变体生成**：模型以特定提示 *P(q, y\*_q)* 为条件，自行生成 *k* 个变体问题；
  3. **在线过滤**：
     - 仅保留难度适中（准确率落在 [τ_low, τ_high]）的样本；
     - 丢弃长度超过阈值 *L_max* 的问题；
  4. 形成高质量动态训练批次，为下阶段优化提供多样化输入。
  
- **理论贡献**：  
  有效防止模型依赖表面模式，通过语义多样化学习问题逻辑结构。

---

### 3. 模块二：测试时混合探索（Test-time Hybrid Exploration）

包含两种更新机制，以平衡模型的“开发与探索”：

- **Intra-Group Exploration (IGE)**：  
  - 对每个样本独立执行 GRPO 强化学习；
  - 针对每个变体计算内部投票奖励；
  - 聚焦于精确解答具体问题（准确性导向）。

- **Cross-Group Exploration (CGE)**：  
  - 跨变体混合采样后统一执行一次多数投票；
  - 跨变体共享伪标签 *y\*_mix*；
  - 促使模型在不同表述间保持一致逻辑（一致性导向）。

两者交替进行，使模型兼顾高精度与可迁移的一致性推理能力。

---

### 4. 强化学习基础

- **优化目标**：  
  \( \max_{\theta} E_{q,o}[R(o, q)] \)
- **算法核心**：基于 GRPO（Group Relative Policy Optimization）：
  - 无需显式 critic；
  - 利用集团内平均奖励归一化优势函数；
  - 优化式：  
    \( L_i(\theta) = \min(\rho_i A_i, clip(\rho_i, 1-ε, 1+ε)A_i) \)。
