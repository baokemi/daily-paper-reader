## 摘要
大型视觉语言模型（LVLMs）在多模态推理方面表现出色，但常常出现幻觉或错误回答，并伴随高置信度，这限制了它们在高风险领域的应用。现有的语言化置信度校准方法主要针对纯文本的大型语言模型（LLMs）开发，通常通过二元答案正确性优化单一整体置信度分数。这种设计与LVLMs不匹配：错误预测可能源于感知失败，也可能在感知正确的情况下由于推理错误，而单一置信度混合了这些来源；同时，视觉不确定性往往受到语言先验的主导。为了解决这些问题，我们提出了VL-Calibration，这是一种强化学习框架，明确地将置信度解耦为视觉置信度和推理置信度。为了在缺乏感知真值标签的情况下监督视觉置信度，我们提出了一种内在视觉确定性估计方法，结合了 (i) 基于图像扰动下的KL散度测量的视觉定位能力，以及 (ii) 基于token熵的内部确定性。我们进一步提出了基于视觉确定性的token级优势重加权方法，以在优化过程中聚焦于相关token，从而抑制无视觉基础的幻觉，同时保留有效的感知。我们在十三个基准上进行了实验，结果表明VL-Calibration能有效改进置信度校准，同时提升视觉推理准确率，并可在不同模型规模和架构间推广至分布外基准。

## Abstract
Large Vision Language Models (LVLMs) achieve strong multimodal reasoning but frequently exhibit hallucinations and incorrect responses with high certainty, which hinders their usage in high-stakes domains. Existing verbalized confidence calibration methods, largely developed for text-only LLMs, typically optimize a single holistic confidence score using binary answer-level correctness. This design is mismatched to LVLMs: an incorrect prediction may arise from perceptual failures or from reasoning errors given correct perception, and a single confidence conflates these sources while visual uncertainty is often dominated by language priors. To address these issues, we propose VL-Calibration, a reinforcement learning framework that explicitly decouples confidence into visual and reasoning confidence. To supervise visual confidence without ground-truth perception labels, we introduce an intrinsic visual certainty estimation that combines (i) visual grounding measured by KL-divergence under image perturbations and (ii) internal certainty measured by token entropy. We further propose token-level advantage reweighting to focus optimization on tokens based on visual certainty, suppressing ungrounded hallucinations while preserving valid perception. Experiments on thirteen benchmarks show that VL-Calibration effectively improves calibration while boosting visual reasoning accuracy, and it generalizes to out-of-distribution benchmarks across model scales and architectures.

## 一、研究背景与核心问题

- **研究背景**：  
  大型视觉语言模型（Large Vision-Language Models, LVLMs）在视觉感知与语言推理任务上表现出强大的综合能力，但常出现“**高置信度的错误输出或幻觉（hallucination）**”。这种错误尤其在医疗或法律等高风险领域造成难以接受的后果。

- **核心问题**：  
  现有置信度校准主要针对纯文本 LLM，通常通过 **单一整体置信度** 预测答案正确性。然而：
  - LVLM 的错误来源复杂：既可能是视觉感知错误，也可能是推理错误；
  - 单一置信度会混合这两类不确定性来源；
  - 视觉不确定性往往被语言先验主导而被掩盖。  
  因此，需要一种能区分“看得准”和“想得准”的置信度表达和校准机制。

## 二、方法论：VL-Calibration 框架

### 1. 核心思想

- **基本目标**：将 LVLM 的整体置信度解耦为“视觉置信度（Visual Confidence）”和“推理置信度（Reasoning Confidence）”；
- **优化方式**：基于强化学习（RL）框架，通过多种奖励机制引导模型学习置信度与实际感知/推理正确性的对齐。

### 2. 模型结构与流程

1. **解耦式输出结构**：
   - 模型生成结构化推理轨迹：
     \[
     τ = (z_{vis}, c_{vis}, z_{reas}, c_{reas}, y)
     \]
     其中：
     - \( z_{vis} \)：视觉阶段描述；
     - \( z_{reas} \)：推理链条；
     - \( c_{vis}, c_{reas} \)：对应置信度；
     - \( y \)：最终答案。
   - 最终整体置信度以调和平均（harmonic mean）融合，确保稳健：
     \[
     Φ(c_{vis}, c_{reas}) = \frac{2c_{vis}c_{reas}}{c_{vis}+c_{reas}}
     \]

2. **视觉确定性估计（Visual Certainty Estimation, VCE）**：
   - 为解决缺乏视觉置信度真值的问题，设计了伪监督指标：
     - **视觉定位性（Visual Grounding）**：在扰动图像（通过随机遮挡）下计算输出分布的 KL 散度，衡量生成内容对视觉输入的依赖；
     - **内部确定性（Internal Certainty）**：计算生成 token 的平均熵值（低熵代表确定性高）；
     - 综合定义视觉确定性分数：
       \[
       S_{vis} = \log(D_{KL} + \epsilon) - \log(H + \epsilon)
       \]

3. **强化学习优化目标**：
   - 奖励函数由三部分组成：
     \[
     R = λ_{acc}R_{acc} + λ_{cal}R_{cal} + λ_{vis}R_{vis}
     \]
     - \(R_{acc}\)：正确性奖励；
     - \(R_{cal}\)：整体置信度与正确性的一致性（Brier损失）；
     - \(R_{vis}\)：视觉置信度与预测视觉确定性一致性；
   - 使用 GRPO（Group-wise Reinforcement Policy Optimization）进行训练。

4. **Token级优势重加权（Token-level Advantage Reweighting, TAR）**：
   - 对视觉描述阶段的每个 token，根据视觉确定性调整 advantage 值；
   - 对低确定性且出错的 token 施加更大惩罚，从而抑制“无根据的幻觉”。

## 七、方法与实验的亮点

- 🔹 **创新性解耦机制**：首次将置信度分离为视觉与推理两部分，能定位不确定性来源。  
- 🔹 **无标签监督创新**：通过 KL 散度 + 熵构造“视觉确定性估计”，无需额外标注。  
- 🔹 **强化学习设计精巧**：多重奖励信号（准确率/校准/视觉置信）形成互补优化。  
- 🔹 **精细级别优化**：token级 advantage 重加权，有效减少幻觉现象。  
- 🔹 **广泛验证**：涵盖13个数据集、不同模型规模与架构，性能提升稳定且显著。  
- 🔹 **可解释性强**：通过热力图和信度曲线展示视觉与推理置信度的显著分离。

## 八、不足与局限

- ⚠ **规模限制**：当前实验仅验证至 30B 模型规模，尚未在 70B+ 等超大模型上验证。  
- ⚠ **计算成本高**：视觉确定性估计需双前向传递，增加约 11% 计算开销。  
- ⚠ **依赖视觉扰动策略**：在极端遮挡或非自然图像场景下，KL 指标可能不稳定。  
- ⚠ **视觉置信度仍为伪标签监督**：可能存偏差风险，尤其在高噪声或多目标视觉输入中。  
- ⚠ **应用推广性需验证**：在真实多模态应用（医疗图像、视频推理）中的实用性尚待评估。
