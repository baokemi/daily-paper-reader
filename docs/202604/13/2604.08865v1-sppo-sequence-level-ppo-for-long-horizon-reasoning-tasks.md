## 摘要
近端策略优化（Proximal Policy Optimization，PPO）在将大型语言模型（Large Language Models，LLMs）与可验证奖励对齐的推理任务中起着核心作用。然而，标准的基于 token 的 PPO 在此场景中表现不佳，原因在于长链式推理（Chain-of-Thought，CoT）过程中时间信用分配的不稳定性以及价值模型的高昂内存成本。尽管无评论器（critic-free）的替代方法如 GRPO 能缓解这些问题，但它们需要多个样本来进行基线估计，从而带来显著的计算开销，严重限制了训练吞吐量。本文提出了序列级 PPO（Sequence-Level PPO，SPPO），一种可扩展的算法，结合了 PPO 的采样效率与基于结果更新的稳定性。SPPO将推理过程重新表述为一个序列级上下文赌博机（Sequence-Level Contextual Bandit）问题，采用解耦的标量价值函数来在不进行多重采样的情况下生成低方差的优势信号。针对数学基准的广泛实验表明，SPPO 显著优于标准 PPO，并与计算密集的基于群组的方法性能相当，为推理型 LLM 的对齐提供了一种资源高效的框架。

## Abstract
Proximal Policy Optimization (PPO) is central to aligning Large Language Models (LLMs) in reasoning tasks with verifiable rewards. However, standard token-level PPO struggles in this setting due to the instability of temporal credit assignment over long Chain-of-Thought (CoT) horizons and the prohibitive memory cost of the value model. While critic-free alternatives like GRPO mitigate these issues, they incur significant computational overhead by requiring multiple samples for baseline estimation, severely limiting training throughput. In this paper, we introduce Sequence-Level PPO (SPPO), a scalable algorithm that harmonizes the sample efficiency of PPO with the stability of outcome-based updates. SPPO reformulates the reasoning process as a Sequence-Level Contextual Bandit problem, employing a decoupled scalar value function to derive low-variance advantage signals without multi-sampling. Extensive experiments on mathematical benchmarks demonstrate that SPPO significantly surpasses standard PPO and matches the performance of computation-heavy group-based methods, offering a resource-efficient framework for aligning reasoning LLMs.

  1. **信用分配不稳定性**：标准 PPO 依赖 token 级评论器（Critic）进行时间差分（TD）学习，但推理任务中的奖励极度稀疏（仅最终结果可验证），导致上千 token 的长时序奖励传递失真，造成优势信号“尾部塌陷”。
  2. **计算与内存成本高昂**：训练大型 LLM 的评论器模型需与策略模型等规模，内存占比过大，限制了在多 GPU 环境下的更新频率。

- **已有方案的局限**  
  虽然无评论器方法如 GRPO（Group Relative Policy Optimization）通过多样本均值归一化缓解了上述问题，但它需要为每个提示样本生成多个响应（例如 N=8），造成 **训练吞吐量显著下降**。

- **研究目标**  
  本文旨在**统一 PPO 的采样效率与 GRPO 的稳定性**，提出一种低方差、低内存开销的强化学习框架，用于资源受限的长程推理任务。

## 二、方法论：Sequence-Level PPO（SPPO）

### 1. 核心思想

- **从 MDP 到 Bandit**  
  作者将推理过程从传统的「token 级马尔可夫决策过程 (MDP)」重新建模为「序列级上下文赌博机 (Sequence-Level Contextual Bandit)」：
  - 将整个回答序列视为一个原子动作；
  - Prompt 作为静态上下文；
  - 奖励仅依据最终结果正确性（0 或 1）。

  这样可以**消除时序信用分配问题**，将长期稀疏奖励压缩为一个单步决策问题。

### 2. 核心机制

- **标量价值函数（Scalar Value Function）**  
  SPPO 引入一个价值模型 \( V_\phi(sp) \)，预测给定 prompt 的成功概率，用二元交叉熵损失训练：
  \[
  L_V(\phi) = -\mathbb{E}[R \log V_\phi(sp) + (1-R)\log(1-V_\phi(sp))]
  \]
  其中 \( R \in \{0, 1\} \) 为最终验证奖励。

- **统一优势估计（Unified Advantage）**  
  序列级优势定义为：
  \[
  A(sp,a) = R - V_\phi(sp)
  \]
  该优势值被**广播到序列中的所有 token**，实现稳定更新，无需多样本基线。

- **策略更新目标**
  改进的 PPO clipped-objective 为：
  \[
  J_{SPPO}(\theta) = \mathbb{E}_{sp,a,t}\left[\min(r_t(\theta)A(sp,a), \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon)A(sp,a))\right]
  \]
  每个 token 的比例项相同优势信号，使得优化与序列长度无关，极大降低了方差。

### 3. 实现策略

- 引入“**解耦评论器（Decoupled Critic）**”：用 1.5B 模型对齐 7B 策略模型，减少内存占用约 12.8%。
- 使用 `verl` 框架进行分布式训练，强化计算稳定性与内存管理。
