## 摘要
近年来，基于视频的人体再识别（ReID）因其能够利用时空线索在非重叠摄像头间匹配个体而备受关注。然而，现有方法在高难度场景（如体育和舞蹈表演）下表现不佳，这些场景中多个人穿着相似的服装并进行动态动作。为克服这些挑战，我们提出了 CG-CLIP，一种由文字描述引导的全新 CLIP 框架，结合了显式文本描述和可学习的 token。该方法包含两个关键组件：Caption-guided Memory Refinement（CMR）和 Token-based Feature Extraction（TFE）。CMR 利用多模态大型语言模型（MLLM）生成的文字描述来细化身份特征，从而捕捉细粒度细节。TFE 通过与固定长度可学习 token 的交叉注意力机制高效聚合时空特征，同时降低计算开销。我们在两个标准数据集（MARS 和 iLIDS-VID）以及两个新构建的高难度数据集（SportsVReID 和 DanceVReID）上进行了评估。实验结果表明，我们的方法在所有基准上均显著优于当前的最新方法。

## Abstract
In recent years, video-based person Re-Identification (ReID) has gained attention for its ability to leverage spatiotemporal cues to match individuals across non-overlapping cameras. However, current methods struggle with high-difficulty scenarios, such as sports and dance performances, where multiple individuals wear similar clothing while performing dynamic movements. To overcome these challenges, we propose CG-CLIP, a novel caption-guided CLIP framework that leverages explicit textual descriptions and learnable tokens. Our method introduces two key components: Caption-guided Memory Refinement (CMR) and Token-based Feature Extraction (TFE). CMR utilizes captions generated by Multi-modal Large Language Models (MLLMs) to refine identity-specific features, capturing fine-grained details. TFE employs a cross-attention mechanism with fixed-length learnable tokens to efficiently aggregate spatiotemporal features, reducing computational overhead. We evaluate our approach on two standard datasets (MARS and iLIDS-VID) and two newly constructed high-difficulty datasets (SportsVReID and DanceVReID). Experimental results demonstrate that our method outperforms current state-of-the-art approaches, achieving significant improvements across all benchmarks.

## 二、方法论与模型设计

### 1. 总体框架：CG-CLIP  
论文提出 **CG-CLIP（Caption-Guided CLIP）** 框架，在CLIP基础上引入“文字描述指导”机制，以显式文本语义辅助视觉特征学习。  
框架包含两个关键模块：

### 2. Caption-guided Memory Refinement（CMR）
- **核心思想**：利用文字描述引导“身份记忆（Memory）”的细化，使模型关注更具辨识力的细节。  
- **主要流程**：
  1. 使用多模态大型语言模型（Phi-4-Multimodal）为每个个体生成详细的文字描述；
  2. 通过CLIP文本编码器提取嵌入，构建每个身份的**文本记忆（Text Memory）MT**；
  3. 将MT作为查询，与图像特征作为键值通过**交叉注意力（Cross-attention）**融合，从图像中选择身份相关patch；
  4. 形成**精炼记忆（Refined Memory MR） = 图像记忆 MI + 文本引导记忆 M̂T**；
  5. 使用MR进行对比学习，将视觉特征与身份保持对齐。  

- **损失函数**：  
  包括视频对记忆对比损失 \( L_{v2rm} \)，三元组损失 \( L_{tri} \)，以及交叉熵损失 \( L_{ce} \)：  
  \[
  L_{total} = L_{v2rm} + L_{tri} + L_{ce}
  \]  
  其中 CMR 仅在训练阶段使用，推理时不增加计算量。

### 3. Token-based Feature Extraction（TFE）
- **目标**：解决自注意力在高帧率或高分辨场景中计算量随输入长度平方增长的问题。  
- **实现机制**：
  1. 采用固定长度的可学习token（Learnable Tokens）；
  2. 通过跨帧的交叉注意力机制聚合时空特征；
  3. 注意力复杂度与输入长度呈线性增长；
  4. 对每帧输出加权聚合后经时间平均池化获得视频级嵌入。  

- **效果**：显著减少显存与计算开销（70%计算量降低），同时保持特征表达充分。

## 七、优点与创新点

1. **语义引导学习**：首次在视频ReID中系统性引入MLLM自动生成文字描述，增强个体辨识的细粒度表征。  
2. **Caption-guided Memory Refinement 模块**：通过跨模态交叉注意力实现图像和文字的身份层级融合，稳定且噪声鲁棒。  
3. **高效的TFE机制**：以线性复杂度聚合时空特征，兼顾性能与速度；对比TF-CLIP在相同GPU中推理时间更短。  
4. **新数据集贡献**：提供两个高难度基准（SportsVReID、DanceVReID），填补体育与舞蹈场景空白。  
5. **实验与可视化全面**：含跨数据集验证、超参敏感分析、t-SNE聚类、注意力热力图，对模型可解释性有显著展示。

## 八、不足与局限

- **依赖文本质量**：模型性能受生成caption的精确度影响显著，若描述含有虚假或模糊信息会削弱CMR效果。  
- **场景限制**：当人物外观几乎完全相同（如舞蹈群体），文字难以区分极细差异，语言表达力存在上限。  
- **计算瓶颈**：虽TFE降低注意力计算，但整体推理仍受限于CLIP主干的复杂度。  
- **可扩展性**：实验集中于ViT-B/16，缺乏在轻量模型或其他模态（如红外、低清画面）上的验证。  
- **真实部署挑战**：未涉及实时多目标追踪等在线ReID场景的时延测试。
