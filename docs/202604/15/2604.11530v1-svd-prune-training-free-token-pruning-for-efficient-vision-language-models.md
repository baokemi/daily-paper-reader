## 摘要
视觉-语言模型（Vision-Language Models, VLM）通过联合处理视觉和文本信息，推动了多模态学习的革新。然而，由于处理长序列视觉 token 带来的高计算和存储需求，它们面临显著挑战。许多现有方法依赖局部启发式指标，如注意力得分或 token 范数。然而，这些标准易受位置偏差和信息分散的影响，限制了在高剪枝率下保留关键信息的能力，导致在视觉细节丰富的图像上性能下降。为了解决这些问题，我们提出了 SVD-Prune，这是一种基于奇异值分解（Singular Value Decomposition, SVD）的免训练、可即插即用的 token 剪枝方法。该方法分解视觉 token 特征矩阵，并利用统计杠杆得分（statistical leverage scores）选择前 K 个 token，从而确保仅保留对全局主导方差贡献最大的 token。实验表明，在极端视觉 token 预算下，SVD-Prune 在性能上持续优于以往的剪枝方法，即使在仅保留 32 或 16 个视觉 token 的情况下仍能保持强劲表现。

## Abstract
Vision-Language Models (VLM) have revolutionized multimodal learning by jointly processing visual and textual information. Yet, they face significant challenges due to the high computational and memory demands of processing long sequences of vision tokens. Many existing methods rely on local heuristics, such as attention scores or token norms. However, these criteria suffer from positional bias and information dispersion, limiting their ability to preserve essential content at high pruning ratios and leading to performance degradation on visually detailed images. To address these issues, we propose SVD-Prune, a trainingfree, plug-and-play token pruning method based on Singular Value Decomposition. It decomposes the vision token feature matrix and selects the top-K tokens using statistical leverage scores, ensuring only tokens contributing most to the dominant global variance are preserved. Experiments show that SVD-Prune consistently outperforms prior pruning methods under extreme vision token budgets, maintaining strong performance even with 32 and 16 vision tokens.

## 2. 方法论与技术细节

### 2.1 核心思想
- 作者提出 **SVD-Prune**，一种基于**奇异值分解 (Singular Value Decomposition, SVD)** 的无训练 token 剪枝算法。
- 关键直觉：  
  通过对视觉特征矩阵进行低秩分解，保留对整体方差贡献最大的主成分，从而找到**信息最具代表性**的 token，而非依赖注意力分布。

### 2.2 算法流程（文字化）

1. **输入：**  
   从视觉编码器获得的特征矩阵 \( F \in \mathbb{R}^{T \times D} \)，其中 T 为视觉 token 数，D 为隐藏维度。

2. **步骤一：SVD 分解**  
   对 F 进行奇异值分解：
   \[
   F = U \Sigma V^T
   \]
   - \( U \)：描述 token 空间的主成分方向；
   - \( \Sigma \)：奇异值矩阵，反映各方向的方差大小。

3. **步骤二：主成分截断**  
   计算累计方差贡献率：
   \[
   c_k = \sum_{i=1}^{k} \frac{s_i^2}{\sum_j s_j^2}
   \]
   选取最小 k 使得累计方差 ≥ ε（例如 ε = 0.9），以保留主要信息。

4. **步骤三：计算 token 杠杆得分（Leverage Scores）**  
   对第 t 个 token：
   \[
   \ell_t = \frac{1}{k} \| U_{t,[1:k]} \|_2^2
   \]
   表示该 token 对主要方差信息的贡献度。

5. **步骤四：确定保留的 token 子集**  
   按杠杆得分从高到低排序，累积求和直至达到整体方差阈值 ε；取对应的 token 索引集，并恢复其原始空间顺序。

6. **输出：**  
   剪枝后的特征矩阵 \( F' \in \mathbb{R}^{T' \times D} \)，其中 \( T' < T \)。

### 2.3 特点
- 无需额外训练或微调；
- 可作为**“plug-and-play”模块**嵌入现有模型；
- 通过全局低秩结构避免局部偏差；
- 保留原始空间顺序，兼容后续视觉-语言推理。
