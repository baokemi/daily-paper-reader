## 摘要
大型视觉语言模型在图像描述中可能会产生对象幻觉，这突显了有效检测和缓解策略的必要性。以往研究通常依赖模型对视觉标记的注意力权重作为检测信号。我们揭示了基于粗粒度注意力的分析由于隐藏的混杂因素（尤其是标记位置和描述中的对象重复）而不可靠。这导致了辛普森悖论：当统计数据被聚合时，注意力趋势会反转或消失。基于这一观察，我们提出了 HaloProbe，这是一种贝叶斯框架，通过对外部描述统计和内部解码信号进行分解来估计标记级的幻觉概率。HaloProbe 使用平衡训练来分离内部证据，并结合对外部特征的学习先验，从而恢复真实的后验分布。尽管基于干预的缓解方法通常通过修改模型内部导致实用性或流畅性下降，我们使用 HaloProbe 作为一种外部评分信号来进行非侵入式缓解。我们的实验表明，HaloProbe 引导的解码比最新的基于干预的方法更有效地减少幻觉，同时保持实用性。

## Abstract
Large vision-language models can produce object hallucinations in image descriptions, highlighting the need for effective detection and mitigation strategies. Prior work commonly relies on the model's attention weights on visual tokens as a detection signal. We reveal that coarse-grained attention-based analysis is unreliable due to hidden confounders, specifically token position and object repetition in a description. This leads to Simpson's paradox: the attention trends reverse or disappear when statistics are aggregated. Based on this observation, we introduce HaloProbe, a Bayesian framework that factorizes external description statistics and internal decoding signals to estimate token-level hallucination probabilities. HaloProbe uses balanced training to isolate internal evidence and combines it with a learned prior over external features to recover the true posterior. While intervention-based mitigation methods often degrade utility or fluency by modifying models' internals, we use HaloProbe as an external scoring signal for non-invasive mitigation. Our experiments show that HaloProbe-guided decoding reduces hallucinations more effectively than state-of-the-art intervention-based methods while preserving utility.

## 二、方法论：HaloProbe 框架

### 1. 核心思想

- 将幻觉检测视为一个**条件概率推断问题**：对每个“对象令牌”（object token）预测其为“正确”或“幻觉”的概率。
- 通过**贝叶斯分解**将内部信号（internal model signals）与外部特征（external caption statistics）解耦，减少混淆和偏置。
- 框架目标是估计真实的后验概率：
  \[
  p(y|x_i, x_e)
  \]
  其中 \(x_i\) 为由 LVLM 内部生成的特征（如注意力分布、解码 logits），\(x_e\) 为外部结构化特征（如 token 位置、对象出现次数、是否首现等）。

### 2. 核心模块

- **Balanced Estimator（平衡估计器）**：
  - 在类别均衡条件下（两个类别概率相等）训练，学习内部特征与标签的相关性。
  - 输出 \( f_\theta(x_i, x_e) \approx p_{bal}(y=1|x_i, x_e) \)。

- **Prior Estimator（先验估计器）**：
  - 在真实（不平衡）分布上单独训练，学习外部特征与标签的先验关系：
    \[
    g_\phi(x_e) = p(y=1|x_e)
    \]
- **后验恢复（Posterior Correction）**：
  - 通过整合两者得到最终概率估计公式：
    \[
    p(y=1|x_i, x_e) = \frac{f_\theta g_\phi}{f_\theta g_\phi + (1-f_\theta)(1-g_\phi)}
    \]
  - 有效地融合内外信号，抑制偏差。

### 3. 应用方式

- **非侵入式检测**：不修改模型内部参数，仅作为一个独立评分器评估生成结果。
- **两种缓解策略**：  
  1. **HaloProbe 引导的 Beam Search**：在 beam search 解码过程中用 HaloProbe 概率对候选生成重新打分，优先选取低幻觉候选。  
  2. **后处理（Post-processing）**：用 GPT-5 作为“修正器”删除 HaloProbe 标记出的幻觉词汇，保持语义和流畅性。

## 六、主要结论与发现

1. **HaloProbe 在检测性能上显著超越现有方法**：  
   LLaVA-1.5 上 ACC=90%、AUROC=93.5%，比 DIML 提升约 5 个百分点。

2. **非侵入式缓解同样卓越**：  
   在多模型、三种解码策略下，HaloProbe+Post-process 与 HaloProbe+Beam 均显著降低 Cs 与 Ci，且不损失 F1。

3. **干预方法副作用明显**：  
   作者实验证实注意力干预导致 decoding 不稳定、重复增多、语言退化，而 HaloProbe 不影响流畅性。

4. **理论发现**：  
   注意力信号与幻觉之间并非简单负相关，存在由位置和重复引起的辛普森悖论；需在条件化处理下分析。

5. **扩展性**：  
   HaloProbe 泛化到 attribute 与 relation 幻觉任务上仍显著提升性能。
