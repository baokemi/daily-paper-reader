## 摘要
社交机器人检测对于在线社交平台的稳定与安全至关重要。然而，当前最先进的机器人检测模型大多是独立开发的，忽视了跨平台共享检测模式所带来的性能提升和快速识别新兴机器人变体的潜力。数据分布和模型架构的异质性进一步增加了设计有效的跨平台、跨模型检测框架的难度。为解决这些挑战，我们提出了 FedRio（个性化联邦社交机器人检测的协作强化对比对抗蒸馏框架）。我们首先为每个客户端引入了一个自适应消息传递模块，作为图神经网络的主干结构。为了高效共享全球数据分布知识，我们设计了基于生成对抗网络的联邦知识提取机制。此外，我们采用多阶段对抗性对比学习策略，以加强客户端间特征空间的一致性，并减少本地模型与全局模型之间的偏差。最后，我们在服务器端采用自适应参数聚合策略，在客户端侧通过基于强化学习的参数控制机制，更好地应对异构联邦环境中的数据异质性。在两个真实社交机器人检测基准上的大量实验表明，FedRio 在检测准确率、通信效率和特征空间一致性方面均明显优于现有的联邦学习基线，并且在更严格的隐私约束下仍能保持与已发表的集中式结果相当的竞争力。

## Abstract
Social bot detection is critical to the stability and security of online social platforms. However, current state-of-the-art bot detection models are largely developed in isolation, overlooking the benefits of leveraging shared detection patterns across platforms to improve performance and promptly identify emerging bot variants. The heterogeneity of data distributions and model architectures further complicates the design of an effective cross-platform and cross-model detection framework. To address these challenges, we propose FedRio (Personalized Federated Social Bot Detection with Cooperative Reinforced Contrastive Adversarial Distillation framework. We first introduce an adaptive message-passing module as the graph neural network backbone for each client. To facilitate efficient knowledge sharing of global data distributions, we design a federated knowledge extraction mechanism based on generative adversarial networks. Additionally, we employ a multi-stage adversarial contrastive learning strategy to enforce feature space consistency among clients and reduce divergence between local and global models. Finally, we adopt adaptive server-side parameter aggregation and reinforcement learning-based client-side parameter control to better accommodate data heterogeneity in heterogeneous federated settings. Extensive experiments on two real-world social bot detection benchmarks demonstrate that FedRio consistently outperforms state-of-the-art federated learning baselines in detection accuracy, communication efficiency, and feature space consistency, while remaining competitive with published centralized results under substantially stronger privacy constraints.

  建立一个隐私保护的、可跨平台协同学习的框架，解决社交机器人检测中跨分布、跨模型的异构学习问题。

## 二、方法概述与核心技术

### 1. 总体框架
提出 **FedRio**（Personalized Federated Social Bot Detection via Cooperative Reinforced Contrastive Adversarial Distillation），其核心思想是：
> 在不共享原始数据的前提下，通过 **对抗生成式知识蒸馏 + 强化学习 + 对比学习** 实现跨平台模型协作与特征一致性。

FedRio 包括四个主要模块：

1. **自适应消息传递模块（Adaptive Message Passing）**  
   - 以 **GraphSAGE** 与 **GIN** 构建两个网络：分别负责“消息接收”和“消息发送”。  
   - 通过 **Gumbel-Softmax 重参数化** 实现可微的节点层级消息传递策略选择。  
   - 公式思想：节点边权由两个动作概率分布决定，边的保留与传播方式根据节点状态动态优化。

2. **联邦对抗对比蒸馏机制（Federated Adversarial Contrastive Knowledge Distillation）**
   - 设计两个分类器（一个全局共享、一个客户端特有）协同学习；
   - 使用全局生成器 **G**（基于 GAN）产生伪样本，避免数据共享；
   - 多阶段优化：
     - 阶段① 分类器对真实与伪数据进行蒸馏；
     - 阶段② 特征提取器通过对抗损失和对比损失维持特征空间对齐；
     - 阶段③ 客户端生成器优化多样性与对抗性，防止模式塌陷；
   - 总体目标函数由分类损失、蒸馏损失、对抗损失、对比损失联合组成。

3. **客户端强化学习更新模块（RL-based Parameter Update）**
   - 将每轮训练视为一个马尔可夫决策过程（MDP）；
   - 采用 **Double Deep Q-Network (DDQN)** 学习客户端参数更新权重；
   - **状态空间**：客户端特征向量 + 预测分布 + 损失值；
   - **动作空间**：选择各客户端的全局参数整合比率；
   - **奖励函数**：根据全局模型验证精度递增给奖励，提高收敛速度。

4. **服务器端自适应参数聚合（Adaptive Parameter Aggregation）**
   - 引入 **可训练神经元级权重矩阵** 进行细粒度聚合；
   - 结合伪数据的交叉熵损失反向更新客户端权重矩阵，提升聚合精度。

2. 每轮通信：

## 三、实验设计

### 1. 使用数据集
- **Vendor-19**：包含约 5,349 账号、11,284 边（含 Bot 与人类节点）。
- **TwiBot-20**：包含 11,826 节点、85,927 边，是当前最大公开 Twitter Bot 数据集。
- 模拟数据异质性：通过 **Dirichlet 分布 Dir(α)** 控制非 IID 程度（α ∈ {1, 0.5, 0.1, 0.05}）。

### 2. 实验场景与基线方法
对比了多个 **数据无关（Data-Free）联邦学习** 方法：
- FedAvg、FedProx（传统训练优化型）
- FedDF、FedDistill（知识蒸馏型）
- FedGen、FedFTG（生成式蒸馏）
- FedACK（基于对抗蒸馏的异构联邦）
并参照集中式社交机器人检测模型（如 SEBot、BotDGT、LMBot）作性能上限的背景比较。

## 六、主要结论与发现

- FedRio 在两个基准数据集上 **平均提升 3%–25% 的检测精度**，尤其在高度非 IID 场景（α ≤ 0.1）下仍保持稳定；
- 通信轮数显著减少：达到指定精度的速度比 FedGen、FedACK 快约 2–10 倍；

- 相比集中式机器人检测模型（需访问全平台数据），FedRio 在私有数据隔离的前提下仍达到集中式模型准确率的约 93%。
