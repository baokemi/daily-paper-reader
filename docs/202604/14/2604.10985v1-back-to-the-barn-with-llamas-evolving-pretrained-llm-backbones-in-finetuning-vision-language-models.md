## 摘要
视觉语言模型（VLMs）通过利用强大的预训练大型语言模型（LLMs）作为核心推理骨干而迅速发展。随着具备更强推理能力、遵循指令能力与泛化能力的新一代LLM不断出现，迫切需要高效地更新现有VLMs以融合这些进展。然而，将新LLM整合到VLM中，尤其是探索演化中的LLM如何影响多模态推理、对齐与任务特定性能，仍然研究不足。鉴于预训练LLM骨干的快速演化，填补这一空白对VLM的发展至关重要。本研究对预训练LLM骨干的变化如何影响VLM下游任务性能进行了受控且系统的研究。通过在基于LLAMA-1、LLAMA-2和LLAMA-3的VLM中保持视觉编码器、训练数据和后训练算法一致，我们发现更新的LLM骨干并不总是带来更好的VLM，性能取决于下游任务类型。例如，在视觉问答任务中，更新的LLM骨干倾向于解决不同类型的问题，而不仅仅是更多问题。我们的分析表明，这种现象源自模型处理信息方式的差异，包括更校准的置信度与更稳定的内部表征。我们还发现，某些VLM能力仅出现在最新一代LLM中，而主要依赖视觉理解的任务从新LLM骨干中获益甚少。

## Abstract
Vision-Language Models (VLMs) have rapidly advanced by leveraging powerful pre-trained Large Language Models (LLMs) as core reasoning backbones. As new and more capable LLMs emerge with improved reasoning, instruction-following, and generalization, there is a pressing need to efficiently update existing VLMs to incorporate these advancements. However, the integration of new LLMs into VLMs, particularly how the evolving LLMs contribute to multimodal reasoning, alignment, and task-specific performance remains underexplored. Addressing this gap is important for VLM development, given the rapid evolution of pretrained LLM backbones. This study presents a controlled and systematic investigation of how changes in the pretrained LLM backbone affect downstream VLM task performance. By having the vision encoder, training data, and post-training algorithm remain same across LLAMA-1, LLAMA-2, and LLAMA-3 based VLMs, we find that newer LLM backbones do not always lead to better VLMs, but the performance depends on the downstream VLM task. For example, in visual question and answering tasks, newer LLM backbones tend to solve different questions rather than just more questions, and our analysis shows this is driven by differences in how the models process information, including better calibrated confidence and more stable internal representations. We also find that some VLM capabilities appear only in the newest LLM generation, while tasks that depend mainly on visual understanding see little benefit from a newer LLM backbone.

## 一、核心问题与研究背景

- **问题起点**：视觉语言模型（Vision-Language Models, VLMs）已成为多模态 AI 的关键，但其核心语言推理能力依赖于预训练大型语言模型（Large Language Models, LLMs）。随着 LLAMA 等 LLM 持续升级（LLAMA-1 → LLAMA-2 → LLAMA-3），研究界尚不清楚：  
  → 新一代 LLM 作为语言骨干时，是否能稳定提升 VLM 的下游性能？  
  → LLM 演化对多模态对齐、推理与表现的影响机制是什么？

- **研究动机**：现有研究对比不同 VLM（如 LLaVA、Qwen-VL、InternVL）时，往往存在视觉编码器、训练数据、实现细节等差异，无法隔离出 LLM 变化的真正影响。因此，作者提出进行**受控实验**，仅更换语言骨干。

- **核心目标**：系统分析 LLM 演化对 VLM 下游任务性能的影响，评估不同任务（推理型、感知型）的响应差异，为未来 VLM 迭代提供实证指导。

## 二、方法论与技术框架

### 1. 总体思路
- 构建一系列架构一致的 VLM（基于 NVIDIA NeMo 框架的 NeVA 模型），只改变语言骨干（LLAMA-1、LLAMA-2、LLAMA-3）。
- 保持视觉编码器、训练数据、微调算法均一致，从而实现**单变量控制实验**。

### 2. 模型组成
- **视觉编码器**：采用固定的 CLIP-ViT-L/14 作为图像特征提取模块。  
- **投影模块（Projector）**：将视觉向量映射到 LLM embedding 空间，实现模态对齐。  
- **语言骨干**：分别使用不同代 LLAMA 模型（1、2、3.1-instruct），作为语言解码器。  

### 3. 微调流程
分为两阶段：
1. **对齐阶段（Warm-up Alignment）**
   - 数据集：*SciCap*（科学图表文本描述），用于学习视觉→文本的基本映射。
   - 优化：同时微调视觉编码器、投影器与 LLM，直到验证集稳定。
2. **监督微调阶段（Supervised Fine-Tuning, SFT）**
   - 使用多领域下游数据集进行针对性训练。
   - 明确采用 SFT 而非强化学习对齐，以确保稳定和可复现。

### 4. 实现细节技术
- 框架：NVIDIA NeMo + NeVA 架构。
- 训练技术：模型并行、激活检查点、AMP O2、Flash Attention 等。
- 所有 LLM checkpoint 均转换为 `.nemo` 格式用于统一训练。

## 三、实验设计与数据集

### 1. 数据与任务
作者选取三类不同性质的下游任务，以覆盖推理、感知、专业领域三类情境。

- 对比对象：LLAMA-1、LLAMA-2、LLAMA-3 三个版本。

## 六、主要结论与发现

1. **性能不随 LLM 进化线性提升**：  
   - ScienceQA 准确率 **下降**（71.8% → 68.4%）；  
   - VQA-Scene **提升**（61.8% → 65.6%）；  
   - Seismic **无显著变化**。  
   → LLM 改进对不同任务有不同影响。

2. **不同任务对改进敏感度不同**：  
   - 记忆驱动推理任务（ScienceQA）不受益；  
   - 视觉语义融合任务（VQA-Scene）显著受益；  
   - 感知主导任务（Seismic）几乎无影响。

3. **新 LLM 改变模型解决问题的类型**：LLAMA-3 解出的题目与旧版完全不同，而不是“更多题”。

4. **新能力出现而非渐进改进**：如地震数据纬度经度预测，LLAMA-3 首次实现数值格式输出（76.5% 准确率），而前两代完全失败。

5. **模型信心与准确率反向变化**：LLAMA-3 准确率更高但置信度更低（0.73 vs 1.0），体现**更好校准**。

6. **内部表征差异显著**：新 LLM 具有更稳定、较小幅度的层间变化，显示出更平滑的信息整合路径。
