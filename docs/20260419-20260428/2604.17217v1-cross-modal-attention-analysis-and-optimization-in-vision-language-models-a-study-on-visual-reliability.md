## 摘要
视觉-语言模型（Vision-Language Models，VLMs）在跨模态任务中表现出色，但近期研究表明，这类模型往往过度依赖文本描述，而未充分利用视觉证据——这一现象被称为“文本捷径学习”（text shortcut learning）。本文提出一种对抗性评估框架，用于量化这种跨模态依赖性，通过测量当语义冲突文本与图像配对不变时的准确率下降（Drop）来实现。我们在一个受控的几何图形数据集（n = 1,000）上应用四种对抗策略：shape_swap、color_swap、position_swap 和 random_text。本文比较了三种配置：基线 CLIP（ViT-B/32）、LoRA 微调以及 LoRA 优化版（集成 Hard Negative Mining、Label Smoothing、分层学习率、余弦重启、课程学习和数据增强）。结果显示，优化模型将平均下降率从 27.5% 降低至 9.8%（相对提升 64.4%，p < 0.001），同时保持 97% 的正常准确率。注意力可视化和嵌入空间分析进一步表明，优化模型更关注视觉特征，并实现了更紧密的跨模态对齐。

## Abstract
Vision-Language Models (VLMs) achieve strong cross-modal performance, yet recent evidence suggests they over-rely on textual descriptions while under-utilizing visual evidence -- a phenomenon termed ``text shortcut learning.'' We propose an adversarial evaluation framework that quantifies this cross-modal dependency by measuring accuracy degradation (Drop) when semantically conflicting text is paired with unchanged images. Four adversarial strategies -- shape\_swap, color\_swap, position\_swap, and random\_text -- are applied to a controlled geometric-shapes dataset ($n{=}1{,}000$). We compare three configurations: Baseline CLIP (ViT-B/32), LoRA fine-tuning, and LoRA Optimized (integrating Hard Negative Mining, Label Smoothing, layer-wise learning rates, Cosine Restarts, curriculum learning, and data augmentation). The optimized model reduces average Drop from 27.5\% to 9.8\% (64.4\% relative improvement, $p{<}0.001$) while maintaining 97\% normal accuracy. Attention visualization and embedding-space analysis confirm that the optimized model attends more to visual features and achieves tighter cross-modal alignment.

- **目标**：系统评估并优化 VLM 的文本依赖程度，使模型能更均衡地利用图像和文本信息，实现更可靠的跨模态理解。

## 二、方法论：核心思想与技术流程

### 1. 对抗评估框架（Adversarial Evaluation Framework）
- **基本思想**：保持图像不变，仅修改文本描述中的关键属性形成语义冲突，测量模型性能下降幅度（Accuracy Drop），以此量化模型的文本依赖度。
- **两阶段流程**：
  1. **正常阶段**：输入原始图像—文本对 \((I, T)\)，得到正常准确率 \(Acc_{normal}\)。
  2. **对抗阶段**：以相同图像配语义冲突文本 \(T'\)，得到 \(Acc_{adv}\)。  
     性能下降：  
     \[
     Drop = Acc_{normal} - Acc_{adv}
     \]
- **度量指标**：
  - 平均下降率 **Avg.Drop**：反映总体文本敏感度。
  - **文本依赖指数（TDI）**：比较“仅文本”与“仅图像”条件下准确率差值，揭示模型对文本的依赖强度。
  - 统计工具：95% Wilson 置信区间、Holm-Bonferroni 多重 t 检验、Cohen’s d 效果量。

### 2. 四种对抗策略
- **形状替换（shape swap）**：更改形状词，如“圆形 → 方形”。
- **颜色替换（color swap）**：更改颜色描述，如“红色 → 蓝色”。
- **位置替换（position swap）**：更改空间位置，如“中心 → 左上”，用于测试空间理解。
- **随机文本（random text）**：完全不相关文本，用作性能下界。

### 3. 模型与优化设计
- **基线模型**：CLIP（ViT-B/32）。
- **LoRA 微调**：在注意力层插入低秩矩阵，实现参数高效适配，仅训练约 0.5% 的参数。
- **LoRA 优化版整合的六项技术**：
  - Hard Negative Mining（困难样本挖掘，30% 比例）
  - Label Smoothing (\(ε = 0.1\))
  - Layer-wise Learning Rate（不同层学习率）
  - Cosine Restarts（余弦退火重启）
  - Curriculum Learning（三阶段课程学习）
  - Data Augmentation（旋转、颜色抖动、模糊、翻转）
  
这些策略形成联合优化机制，综合提高视觉可靠性。
