## 摘要
尽管近期取得了显著进展，视觉-语言编码器仍面临两个核心限制：（1）语言与密集视觉特征之间的对齐较弱，这会影响如开放词汇语义分割等任务；（2）细粒度视觉表征所需的高标记数量，限制了模型在长视频上的可扩展性。本研究针对这两个问题提出了相应的解决方案。我们提出 T-REN（Text-aligned Region Encoder Network，文本对齐区域编码网络），这是一种高效的编码器，可将视觉数据映射为一组紧凑的、与文本对齐的区域级表征（即区域标记）。T-REN 通过在冻结的视觉主干网络上添加一个轻量级网络实现，该网络被训练用来在每个语义区域内池化补丁级表示为区域标记，并将其与区域级文本注释对齐。与视觉-语言主干相比，T-REN 仅增加约 3.7% 的参数，但显著增强了密集跨模态理解能力，同时使标记数量减少几个数量级。具体而言，T-REN 在 ADE20K 开放词汇分割上提升了 +5.9 mIoU，在 COCO 对象级文本-图像检索上提升了 +18.4% 召回率，在 Ego4D 视频对象定位上提升了 +15.6% 召回率，在 VSPW 视频场景解析上提升了 +17.6% mIoU，同时与基于补丁的视觉语言主干相比，图像标记数量减少超过 24 倍、视频标记数量减少超过 187 倍。代码与模型已在 https://github.com/savya08/T-REN 上提供。

## Abstract
Despite recent progress, vision-language encoders struggle with two core limitations: (1) weak alignment between language and dense vision features, which hurts tasks like open-vocabulary semantic segmentation; and (2) high token counts for fine-grained visual representations, which limits scalability to long videos. This work addresses both limitations. We propose T-REN (Text-aligned Region Encoder Network), an efficient encoder that maps visual data to a compact set of text-aligned region-level representations (or region tokens). T-REN achieves this through a lightweight network added on top of a frozen vision backbone, trained to pool patch-level representations within each semantic region into region tokens and align them with region-level text annotations. With only 3.7% additional parameters compared to the vision-language backbone, this design yields substantially stronger dense cross-modal understanding while reducing the token count by orders of magnitude. Specifically, T-REN delivers +5.9 mIoU on ADE20K open-vocabulary segmentation, +18.4% recall on COCO object-level text-image retrieval, +15.6% recall on Ego4D video object localization, and +17.6% mIoU on VSPW video scene parsing, all while reducing token counts by more than 24x for images and 187x for videos compared to the patch-based vision-language backbone. The code and model are available at https://github.com/savya08/T-REN.

## 二、方法论概述

### 1. 核心思想  
提出 **T-REN（Text-aligned Region Encoder Network）**，在冻结的预训练视觉–语言主干（如 DINOv3-based dino.txt）上新增一个轻量模块，将补丁特征聚合为**语义一致的区域 token**并与文本区域注释对齐。最终输出的区域 token 兼具紧凑性与语义丰富度。

### 2. 核心结构与流程  
- **输入**：图像或视频帧，经冻结的 DINOv3 ViT-L/16 编码为补丁特征。  
- **点提示（Point Prompts）**：在图像上布置一个二维网格，每个点作为跨注意力查询。  
- **多 token 生成**：
  - 每个点经提示编码器（Prompt Encoder）生成 \(k=3\) 个查询；
  - 经过两层解码器（包含跨注意力 + 自注意力）与补丁特征交互，获得区级聚合信息；
  - 单头跨注意力进一步汇聚补丁特征形成区域 token。  
- **合并策略**：
  - 使用余弦相似度 (\(>0.975\)) 或区域掩码 IoU (\(>0.8\)) 进行冗余区域 token 合并；
  - 自适应生成最终的 M 个区域 token，随场景复杂度自动调整。  
- **文本对齐**：
  - 使用一个 2 层 MLP 将区域 token 投射至文本嵌入空间；
  - 与文本编码器输出通过余弦相似度对齐，实现跨模态语义对应。  
- **时序聚合（视频场景）**：
  - 连续帧间基于相似度阈值 (\(\tau_{track}=0.65\)) 合并对应目标形成“轨迹 token (track tokens)”；
  - 通过贪婪一对一匹配维持时序一致性。  

### 3. 训练目标  
联合优化四类损失函数以平衡语义一致性与对齐能力：  
- **视觉对比损失 \(L_{cont}^v\)**：使来自同一真实区域的视觉 token 聚合，异类分离。  
- **文本对齐对比损失 \(L_{cont}^t\)**：强化区域 token 与对应文本嵌入的匹配。  
- **蒸馏损失 \(L_{dist}\)**：约束学习到的特征不偏离冻结主干的表征空间。  
- **注意力监督 \(L_{attn}\)**：强制跨注意力掩码接近真实分割掩码。  

### 4. 训练数据与优化  
- 训练集由五个语义分割数据集混合：**COCOStuff、OpenImagesV7、PhraseCut、Mapillary Vistas、SA-1B**。  
- 每张图像采样 128 个点提示，按照区域重叠度加权采样。  
- 使用 **AdamW** 优化器，学习率 0.001，60,000 次迭代（约 <1 epoch），批大小 16。
