## 摘要
近年来，基于视频的人体重识别（video-based person Re-Identification，ReID）因其能够利用时空线索在非重叠摄像机之间匹配个体而受到广泛关注。然而，现有方法在高难度场景（如体育和舞蹈表演）中表现不佳，此类场景中多名个体穿着相似服装并进行动态动作。为应对这些挑战，我们提出了一种新颖的基于文字描述引导的 CLIP 框架——CG-CLIP（Caption-Guided CLIP），该框架结合了显式文本描述与可学习的 token。我们的方法包含两个关键组件：Caption-guided Memory Refinement（CMR，文字描述引导的记忆优化）和 Token-based Feature Extraction（TFE，基于 token 的特征提取）。CMR 利用多模态大语言模型（Multi-modal Large Language Models，MLLMs）生成的文字描述来优化身份特征，捕捉细粒度信息；TFE 则采用基于固定长度可学习 token 的交叉注意力机制，以高效聚合时空特征，降低计算成本。我们在两个标准数据集（MARS 和 iLIDS-VID）以及两个新构建的高难度数据集（SportsVReID 和 DanceVReID）上对该方法进行了评估。实验结果表明，我们的方法在所有基准测试上均优于当前最新方法，取得了显著性能提升。

## Abstract
In recent years, video-based person Re-Identification (ReID) has gained attention for its ability to leverage spatiotemporal cues to match individuals across non-overlapping cameras. However, current methods struggle with high-difficulty scenarios, such as sports and dance performances, where multiple individuals wear similar clothing while performing dynamic movements. To overcome these challenges, we propose CG-CLIP, a novel caption-guided CLIP framework that leverages explicit textual descriptions and learnable tokens. Our method introduces two key components: Caption-guided Memory Refinement (CMR) and Token-based Feature Extraction (TFE). CMR utilizes captions generated by Multi-modal Large Language Models (MLLMs) to refine identity-specific features, capturing fine-grained details. TFE employs a cross-attention mechanism with fixed-length learnable tokens to efficiently aggregate spatiotemporal features, reducing computational overhead. We evaluate our approach on two standard datasets (MARS and iLIDS-VID) and two newly constructed high-difficulty datasets (SportsVReID and DanceVReID). Experimental results demonstrate that our method outperforms current state-of-the-art approaches, achieving significant improvements across all benchmarks.

  1. **细粒度特征难以捕获**：如发型、球衣号码、鞋子颜色等微小差异；

- **论文目标：**  
  提出一种新型“文本引导的视觉-语言融合”框架，通过显式语言描述辅助视频特征学习，从而提升模型在高相似场景下的判别力与效率。

## 2. 提出的方法论：CG-CLIP 框架

### 核心思想
- 将自然语言描述（caption）与视频图像特征融合引导训练，利用多模态大语言模型（MLLM）生成视频中行人的属性描述；
- 在训练期间使用这些文本特征引导视觉特征提取与对比学习，而推理阶段无需文本，因此保持高效。

### 架构组成
**CG-CLIP（Caption-Guided CLIP）**包含两个核心模块：

#### (1) Caption-guided Memory Refinement（CMR）
- **目标：** 在特征对比学习中加入文本语义引导，优化身份记忆（Identity Memory）。
- **机制：**
  - 利用冻结的 CLIP 文本编码器，将 MLLM 生成的文字描述嵌入成一组“文本记忆”（Text Memory），每个身份一个语义原型；
  - 将训练图像特征（Image Memory）通过交叉注意力与 Text Memory 融合；
  - 得到“Refined Memory” \( M_R = M_I + \hat{M}_T \)，结合语义与视觉要素；
  - 使用基于余弦相似度的**视频到记忆对比损失**：
    \[
    L_{v2rm} = -\sum_{p\in D(y_i)} \log \frac{\exp(s(b_p,M_{R_{y_i}}))}{\sum_j \exp(s(b_j,M_{R_{y_i}}))}
    \]
- **优点：**  
  有效注入细粒度身份差异，抵抗 MLLM 文字噪声，同时不引入推理阶段开销。

#### (2) Token-based Feature Extraction（TFE）
- **目标：** 以线性复杂度聚合视频的时空信息。
- **机制：**
  - 引入固定数量的**可学习 tokens（\(Q_a\)）**，作为跨帧特征的查询；
  - 先在时间维度进行交叉注意力（frame-level aggregation），再在空间维度执行 cross/self-attention；
  - 通过平均聚合和 Temporal Average Pooling（TAP）输出时序特征；
  - 该设计使复杂度与输入长度线性相关，大幅节省计算资源。

### 训练策略与损失函数
- 总损失：
  \[
  L_{total} = L_{v2rm} + L_{tri} + L_{ce}
  \]
  其中包括：
  - 视频到记忆对比损失；
  - 三元组损失 (Triplet Loss)；
  - 标签平滑交叉熵损失。

- **推理阶段：** 不使用文字或 CMR，仅依赖视觉编码器 + TFE 输出最终表示。

## 6. 主要结论与发现

- **性能提升显著：**
  - 在 MARS 上：mAP 89.8%，Rank-1 92.5%，超越 TF-CLIP (89.4%)；
  - 在 iLIDS-VID 上：Rank-1 达 96.7%，领先 5.4%；
  - 在高难度 SportsVReID / DanceVReID 上：分别达 mAP 77.7%、Rank-1 90.4% 和 mAP 53.8%、Rank-1 76.0%，均为新 SOTA。
- **泛化强度优异：** 在少量训练数据（20%）时仍领先 1.2% mAP；
- **效率提升显著：** TFE 在相似精度下具更低延时，更适合实时应用；
- **视觉可解释性增强：** 注意力图验证模型更集中于发型、球衣号、鞋色等关键细粒度部分。

## 7. 优点与亮点

- **原创性高：**
  - 首次在视频 ReID 中引入**显式字幕级文本指导**；
  - 提出 **Memory-level 融合** 而非实例级（instance-wise）对齐的新思路。
- **算力友好：**  
  TFE 模块通过交叉注意力与固定长度 token 实现线性复杂度聚合。
- **通用性强：**  
  训练时需文本，但推理阶段纯视觉即可，兼顾准确性与效率。
- **数据贡献：**  
  发布两个高难度数据集（SportsVReID、DanceVReID），促进该方向公共研究。
- **可解释性良好：**  
  Attention 可视化清晰揭示模型关注关键身份特征。

## 8. 不足与局限性

- **对字幕质量敏感：**  
  MLLM 会出现“幻觉式生成”或错误描述，可能降低效果。
- **应用局限：**
  - 对极端相似或无法用语言表达的差异仍无能为力（例如极度相同服饰的舞蹈演员）；
  - 模型仍依赖大规模 CLIP 编码器，推理延迟与参数量尚不适合实时嵌入设备。
- **数据偏差风险：**  
  新数据集规模有限，主要集中于体育与舞蹈场景；
  未来需在更广泛视频类型上验证鲁棒性。
- **未来改进方向：**
  - 改进字幕质量评估与过滤；
  - 引入面部特征或动作序列建模；
  - 研究轻量化版本以满足实时追踪任务。
