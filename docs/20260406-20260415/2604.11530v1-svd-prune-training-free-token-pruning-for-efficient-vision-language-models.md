## 摘要
视觉语言模型（Vision-Language Models，VLM）通过联合处理视觉与文本信息，革新了多模态学习。然而，它们在处理视觉 token 的长序列时面临着巨大的计算和存储挑战。现有的许多方法依赖于局部启发式策略，如注意力分数或 token 范数。然而，这些标准受到位置偏差和信息分散的影响，限制了它们在高剪枝比下保留关键信息的能力，并导致在视觉细节丰富的图像上性能下降。为解决这些问题，我们提出了 SVD-Prune，这是一种无需训练、即插即用的基于奇异值分解（Singular Value Decomposition）的 token 剪枝方法。该方法分解视觉 token 特征矩阵，并利用统计杠杆分数（statistical leverage scores）选取前 K 个 token，确保仅保留对主导全局方差贡献最大的 token。实验表明，在极端视觉 token 预算下，SVD-Prune 始终优于现有的剪枝方法，即使仅使用 32 或 16 个视觉 token，仍能保持较强的性能。

## Abstract
Vision-Language Models (VLM) have revolutionized multimodal learning by jointly processing visual and textual information. Yet, they face significant challenges due to the high computational and memory demands of processing long sequences of vision tokens. Many existing methods rely on local heuristics, such as attention scores or token norms. However, these criteria suffer from positional bias and information dispersion, limiting their ability to preserve essential content at high pruning ratios and leading to performance degradation on visually detailed images. To address these issues, we propose SVD-Prune, a trainingfree, plug-and-play token pruning method based on Singular Value Decomposition. It decomposes the vision token feature matrix and selects the top-K tokens using statistical leverage scores, ensuring only tokens contributing most to the dominant global variance are preserved. Experiments show that SVD-Prune consistently outperforms prior pruning methods under extreme vision token budgets, maintaining strong performance even with 32 and 16 vision tokens.

## 1. 核心问题与研究背景

- **研究动机**  
  视觉语言模型（VLM）通过同时处理图像与文本信息，已经成为多模态学习的重要方向。然而，这类模型在应用中面临两大瓶颈：
  1. **视觉 token 数量庞大**：典型模型（如 LLaVA 系列）将输入图像划分为数百个视觉 token，导致计算、内存和推理延迟显著增加。
  2. **视觉信息冗余严重**：研究发现，大部分视觉 token 对最终的多模态推理贡献有限。

- **核心问题**  
  如何**在不重新训练模型的条件下**，高效地剪除冗余视觉 token，从而**减轻推理负担，同时尽量保持任务性能**。

- **背景局限**  
  现有剪枝方法多依据局部启发式指标（如注意力权重、token 范数或跨模态相似度），但这些指标存在：
  - **位置偏差（positional bias）**：注意力随序列位置系统性衰减；
  - **信息分散**：局部度量难以捕获全局视觉结构；
  - 导致高剪枝率时性能急剧下降。

## 2. 方法论：SVD-Prune 的核心思想与技术流程

### 核心思想
**SVD-Prune** 通过对视觉 token 特征矩阵执行**奇异值分解 (SVD)**，从**全局结构**出发判断每个 token 的重要性。  
它利用**统计杠杆分数（leverage scores）**衡量 token 对主成分方差的贡献，从而保留最具代表性的少量 token，而无需额外训练。

### 主要步骤与算法描述

1. **输入表示**  
   图像经视觉编码器处理后生成特征矩阵 \( F \in \mathbb{R}^{T \times D} \)，其中：
   - \( T \)：原始视觉 token 数（如 576）；
   - \( D \)：特征维度。

2. **奇异值分解（SVD）**  
   \[
   F = U \Sigma V^\top
   \]
   - \( U \)：左奇异向量矩阵，描述 token 空间的主方向；
   - \( \Sigma \)：奇异值对角矩阵，反映每个主方向的方差；
   - 仅保留前 \( k \) 个主分量以覆盖主要信息。

3. **主方差选择**  
   计算累计方差比：
   \[
   c_k = \frac{\sum_{i=1}^k s_i^2}{\sum_{j=1}^r s_j^2}
   \]
   选取最小的 \( k \) 使得 \( c_k \geq \varepsilon \)，
   - 其中 \( \varepsilon \in [0.7,0.95] \) 表示保留的全局方差比例。

4. **统计杠杆分数计算**  
   每个 token 的重要性定义为：
   \[
   \ell_t = \frac{1}{k} \sum_{j=1}^k (U_{t,j})^2
   \]
   表示第 t 个 token 在主成分空间中的平均投影强度。

5. **剪枝与重排序**  
   - 按杠杆分数降序排序；
   - 选择累积权重达到阈值 \( \varepsilon \) 的最小 token 子集；
   - 恢复空间顺序以保持位置一致性；
   - 输出剪枝后的新矩阵 \( F' \)。

**关键特性**：
- 无需模型改动或重训练；
- 不依赖注意力机制，无位置偏差；
- 可直接嵌入任意视觉-语言模型流水线中。

- **总体结论**：
